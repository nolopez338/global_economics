	\SubsectionBox{Criteria assessment}\vspace{-0.25cm}
	Each assessment criterion is evaluated through the four required implementations. A criterion is considered passed when it is correctly applied in its two required implementations.
	
	For a set of tasks to count as valid and complete, it must include:
	\begin{itemize}
		\item Exactly 2 proportion-test implementations for Criterion C4 and exactly 2 mean-test implementations for Criterion C5
		\item For C4 and C5, one implementation designed so that the expected statistical decision is to reject \(H_0\), and one implementation designed so that the expected statistical decision is to not reject \(H_0\)
		\item At least 1 implementation based on real-world data collection, such as a survey, direct observation, counting, timing, or recording measurements. At least 1 implementation based on web databases or publicly available online datasets
		\item enough evidence of the data-obtention process to verify the work, such as tables, screenshots, survey forms, observation records, database references, or clearly organized notes
	\end{itemize}
	
	\vspace{0.25cm}
	\SubsectionBox{Student Problems}\vspace{-0.25cm}
	
	Use the following shared instructions for all six problems. From the six contexts below, complete exactly 4 implementations in total, following all validity rules above.
	\begin{itemize}
		\item \textbf{(C4--C5)} For each implementation: describe the context; state whether it is a population proportion test or a population mean test; define the parameter and the benchmark claim; explain how the data will be obtained; present the actual sample data or summary sample information; show evidence of the data-obtention process; choose a significance level; write \(H_0\) and \(H_a\) with correct notation; perform the hypothesis-test calculations; state the final statistical decision; and write the conclusion in context.
	\end{itemize}
	
	\begin{enumerate}
		\item \textbf{Problem 1 --- School context}\\
		Create a hypothesis test about a measurable feature of your school.
		
		\item \textbf{Problem 2 --- Home context}\\
		Create a hypothesis test about a measurable feature of life at home.
		
		\item \textbf{Problem 3 --- Neighborhood context}\\
		Create a hypothesis test about a measurable feature of your neighborhood.
		
		\item \textbf{Problem 4 --- Your Grade 11 group context}\\
		Create a hypothesis test about a measurable feature of your own Grade 11 group.
		
		\item \textbf{Problem 5 --- Personal context}\\
		Create a hypothesis test about a measurable feature related to yourself.
		
		\item \textbf{Problem 6 --- Bogotá context}\\
		Create a hypothesis test about a measurable feature of Bogotá that can be investigated at your level.
	\end{enumerate}
