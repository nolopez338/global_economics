\SubsectionBox{Background from C1--C9}
In C1--C7, you learned the full hypothesis-testing process. In C8, you built an interactive normal-curve webpage to visualize p-values. In C9, you added CSV upload and automatic two-group hypothesis testing. In C10, you will apply that tool to three datasets and produce a formal evidence-based article.

\SubsectionBox{Task overview}
Using your C8--C9 webpage, analyze \textbf{three required CSV datasets} and write one professional report in article format. This is an activity sheet; do \textbf{not} submit solved examples or copied model answers.

\SubsectionBox{Required datasets}
Use the following files (simulated for educational purposes):
\begin{itemize}
\item \texttt{G11\_T3\_L4\_C10\_dataset\_1.csv}: Study hours comparison between two Grade 11 groups.
\item \texttt{G11\_T3\_L4\_C10\_dataset\_2.csv}: Delivery times comparison between two service providers.
\item \texttt{G11\_T3\_L4\_C10\_dataset\_3.csv}: Daily exercise minutes comparison between weekday and weekend groups.
\end{itemize}
Each file contains a clear group variable and one numerical variable suitable for two-group hypothesis testing.

\SubsectionBox{Hypothesis-testing requirements (complete for each dataset)}
\begin{enumerate}
\item Upload the CSV into your C8--C9 webpage.
\item Identify the two groups being compared.
\item Identify the variable being tested.
\item State the research question.
\item Define \(H_0\) and \(H_a\).
\item Identify test direction (left-tailed, right-tailed, or two-tailed).
\item Select and justify a significance level \(\alpha\).
\item Use the webpage to calculate the test statistic and p-value.
\item Use the interactive graph to show the p-value region.
\item Make a decision about \(H_0\).
\item Write a contextual conclusion based on your decision.
\item Include a screenshot of the webpage output.
\item Explain what the result means in the real-world context.
\end{enumerate}

\SubsectionBox{Report requirements (article format)}
Your final report must include:
\begin{enumerate}
\item Title
\item Abstract
\item Introduction
\item Methodology
\item Data description
\item Hypothesis-test results for all three datasets
\item Discussion
\item Limitations
\item Conclusion
\item References (APA style)
\item Appendix (screenshots and dataset descriptions)
\end{enumerate}
Do not include copied template text as your final report body; write your own academic-style content.

\SubsectionBox{APA citation requirements}
\begin{itemize}
\item Include at least \textbf{three sources}.
\item Use APA in-text citations when making claims about the context.
\item Include a \textbf{References} section at the end.
\item Do not cite sources that were not used in your report.
\item Use reliable sources (official statistics sites, academic publications, institutional reports, or reputable data portals).
\item Citations are required for \textbf{contextual claims}, not for the webpage's numeric calculations.
\end{itemize}

\SubsectionBox{Deliverables}
Submit all of the following:
\begin{enumerate}
\item Professional report as PDF.
\item Source document used to write the report.
\item The three CSV files used.
\item Screenshots of webpage results for each dataset.
\item Short appendix explaining how the webpage was used.
\item Final webpage files if modified after C8--C9.
\end{enumerate}

\SubsectionBox{Evaluation checklist (validation requirements)}
Your activity is valid only if all checks pass:
\begin{enumerate}
\item All three datasets are analyzed.
\item Each analysis uses the webpage created in C8--C9.
\item Each case includes hypotheses, test direction, p-value, decision, and contextual conclusion.
\item The p-value graph is included through screenshots.
\item The report follows a professional article format.
\item APA-style citations and references are included.
\item Conclusions are consistent with statistical decisions.
\end{enumerate}
