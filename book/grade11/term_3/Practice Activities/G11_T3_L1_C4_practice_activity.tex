\section{Export compliance rate in a logistics hub}
A logistics hub reports that at least 92\% of export containers pass customs compliance checks on the first inspection. A random sample of \(n=250\) containers shows \(x=219\) passes. At \(\alpha=0.05\), test whether the true pass proportion is lower than 0.92.

\subsection*{C4}
\begin{itemize}
    \item \textbf{Hypotheses:} \(H_0\colon p=0.92\), \(H_1\colon p<0.92\).
    \item \textbf{Conditions:} \(np_0=250(0.92)=230\ge 10\), \(n(1-p_0)=250(0.08)=20\ge 10\).
    \item \textbf{Sample proportion:} \(\hat p=219/250=0.876\).
    \item \textbf{Test statistic:}
    \[
    z=\frac{0.876-0.92}{\sqrt{\frac{0.92(0.08)}{250}}}\approx -2.57.
    \]
    \item \textbf{\(p\)-value:} \(P(Z\le -2.57)\approx 0.0051\).
    \item \textbf{Decision:} Reject \(H_0\) because \(0.0051<0.05\).
    \item \textbf{Conclusion:} There is sufficient evidence that the first-pass compliance rate is below 92\%.
\end{itemize}

    \item \textbf{Sample proportion:} \(\hat p=189/300=0.63\).
    \item \textbf{Test statistic:}
    \[

    \item \textbf{Decision:} Reject \(H_0\) because \(0.0401<0.10\).

\section{Defect proportion for exported microchips}
A manufacturer states that the defect proportion of exported microchips is \(p=0.03\). In \(n=500\) chips, inspectors find \(x=25\) defects. At \(\alpha=0.01\), test whether the defect proportion differs from 0.03.

\subsection*{C4}
\begin{itemize}
    \item \textbf{Hypotheses:} \(H_0\colon p=0.03\), \(H_1\colon p\ne 0.03\).
    \item \textbf{Conditions:} \(np_0=15\ge 10\), \(n(1-p_0)=485\ge 10\).
    \item \textbf{Sample proportion:} \(\hat p=25/500=0.05\).
    \item \textbf{Test statistic:}
    \[
    z=\frac{0.05-0.03}{\sqrt{\frac{0.03(0.97)}{500}}}\approx 2.62.
    \]
    \item \textbf{\(p\)-value:} \(2P(Z\ge 2.62)\approx 0.0088\).
    \item \textbf{Decision:} Reject \(H_0\) because \(0.0088<0.01\).
    \item \textbf{Conclusion:} There is sufficient evidence that the defect proportion is not 3\%.
\end{itemize}
