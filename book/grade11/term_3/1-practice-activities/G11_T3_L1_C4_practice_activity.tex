\section{General Problem}
A benchmark claim may refer to either a population proportion or a population mean. In either case, a hypothesis test is conducted at a given significance level \(\alpha\) to evaluate whether the sample evidence supports or contradicts the benchmark value.

\textbf{Proportion template (abstract):} A benchmark percentage (for example, 62\%) is given for a population proportion. A sample of size \(n\) (for example, 200) is collected, with observed successes \(x\) (for example, 112). Using a significance level (for example, 5\%), test whether the true population proportion is lower than, higher than, or different from the benchmark percentage.

\textbf{Mean template (abstract):} A benchmark mean value (for example, 75) is given for a population mean. A sample of size \(n\) (for example, 36) is collected with sample mean \(\bar x\) (for example, 79). If provided, the population standard deviation \(\sigma\) is known (for example, 12). Using a significance level (for example, 1\%), test whether the true population mean is lower than, higher than, or different from the benchmark mean.

\textbf{Questions}
\begin{enumerate}
    \item[(a)] Write the hypotheses for the case where the true parameter is lower than the benchmark.
    \item[(b)] Write the hypotheses for the case where the true parameter is higher than the benchmark.
    \item[(c)] Write the hypotheses for the case where the true parameter is different from the benchmark.
\end{enumerate}

\subsection*{C4}
\begin{itemize}
    \item \textbf{General setup:} Identify the parameter of interest, the benchmark value, the sample information, and the significance level \(\alpha\).
    \item \textbf{Proportion test (one-sample \(z\)-test):}
    \begin{itemize}
        \item Define \(p\) as the true population proportion and \(p_0\) as the benchmark proportion.
        \item State hypotheses according to direction:
        \[
        H_0\colon p=p_0,\qquad H_a\colon p<p_0\ \text{or}\ p>p_0\ \text{or}\ p\ne p_0.
        \]
        \item Check large-sample conditions using \(n\) and \(p_0\): \(np_0\ge 10\) and \(n(1-p_0)\ge 10\).
        \item Compute the sample proportion: \(\hat p=\dfrac{x}{n}\).
        \item Compute the test statistic:
        \[
        z=\frac{\hat p-p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}.
        \]
        \item Compute the \(p\)-value from the standard normal distribution using the tail type (left-tailed, right-tailed, or two-tailed).
        \item Decision rule: reject \(H_0\) if \(p\)-value \(\le \alpha\); otherwise fail to reject \(H_0\).
        \item State the conclusion in words about the population proportion relative to the benchmark.
    \end{itemize}
    \item \textbf{Mean test with known \(\sigma\) (one-sample \(z\)-test):}
    \begin{itemize}
        \item Define \(\mu\) as the true population mean and \(\mu_0\) as the benchmark mean.
        \item State hypotheses according to direction:
        \[
        H_0\colon \mu=\mu_0,\qquad H_a\colon \mu<\mu_0\ \text{or}\ \mu>\mu_0\ \text{or}\ \mu\ne \mu_0.
        \]
        \item Compute the test statistic:
        \[
        z=\frac{\bar x-\mu_0}{\sigma/\sqrt{n}}.
        \]
        \item Compute the \(p\)-value from the standard normal distribution using the tail type (left-tailed, right-tailed, or two-tailed).
        \item Decision rule: reject \(H_0\) if \(p\)-value \(\le \alpha\); otherwise fail to reject \(H_0\).
        \item State the conclusion in words about the population mean relative to the benchmark.
    \end{itemize}
\end{itemize}

\section{Export compliance rate in a logistics hub}
A logistics hub reports that at least 92\% of export containers pass customs compliance checks on the first inspection. In a random sample of 250 containers, 219 pass on the first inspection. Using a significance level of 5\%, test whether the true first-pass proportion is lower than 92\%.

\subsection*{C4}
\begin{itemize}
    \item \textbf{Hypotheses:} \(H_0\colon p=0.92\), \(H_1\colon p<0.92\).
    \item \textbf{Conditions:} \(np_0=250(0.92)=230\ge 10\), \(n(1-p_0)=250(0.08)=20\ge 10\).
    \item \textbf{Sample proportion:} \(\hat p=219/250=0.876\).
    \item \textbf{Test statistic:}
    \[
    z=\frac{0.876-0.92}{\sqrt{\frac{0.92(0.08)}{250}}}\approx -2.57.
    \]
    \item \textbf{\(p\)-value:} \(P(Z\le -2.57)\approx 0.0051\).
    \item \textbf{Decision:} Reject \(H_0\) because 0.51\% is less than 5\%.
    \item \textbf{Conclusion:} There is sufficient evidence that the first-pass compliance rate is below 92\%.
\end{itemize}
