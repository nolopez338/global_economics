	\section{General Problem}
	A benchmark claim may refer to a population proportion. A hypothesis test is conducted at a given significance level \(\alpha\) to evaluate whether the sample evidence supports or contradicts the benchmark value.
	
	\textbf{Proportion template:} A benchmark percentage (for example, 62\%) is given for a population proportion. A sample of size \(n\) (for example, 200) is collected, with observed successes \(x\) (for example, 112). Using a significance level (for example, 5\%), test whether the true population proportion is lower than, higher than, or different from the benchmark percentage.
	
	\textbf{Questions}
	\begin{enumerate}
	    \item[(a)] Write the hypotheses for the case where the true parameter is lower than the benchmark.
	    \item[(b)] Write the hypotheses for the case where the true parameter is higher than the benchmark.
	    \item[(c)] Write the hypotheses for the case where the true parameter is different from the benchmark.
	\end{enumerate}
	
	\subsection*{C4}
	\begin{itemize}
	    \item \textbf{General setup:} Identify the parameter of interest, the benchmark value, the sample information, and the significance level \(\alpha\).
	    \item \textbf{Proportion test (one-sample \(z\)-test):}
	    \begin{itemize}
	        \item Define \(p\) as the true population proportion and \(p_0\) as the benchmark proportion.
	        \item State hypotheses according to direction:
	        \[
	        H_0\colon p=p_0,\qquad H_a\colon p<p_0\ \text{or}\ p>p_0\ \text{or}\ p\ne p_0.
	        \]
	        \item Check large-sample conditions using \(n\) and \(p_0\): \(np_0\ge 10\) and \(n(1-p_0)\ge 10\).
	        \item Compute the sample proportion: \(\hat p=\dfrac{x}{n}\).
	        \item Compute the test statistic:
	        \[
	        z=\frac{\hat p-p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}.
	        \]
	        \item Compute the \(p\)-value from the standard normal distribution using the tail type (left-tailed, right-tailed, or two-tailed).
	        \item Decision rule: reject \(H_0\) if \(p\)-value \(\le \alpha\); otherwise fail to reject \(H_0\).
	        \item State the conclusion in words about the population proportion relative to the benchmark.
	    \end{itemize}
	
	\end{itemize}
	
	\section{Food delivery completion rate}
	A local delivery app states that 64\% of assigned orders are completed without cancellation. In a random sample of 220 assigned orders, 129 are completed without cancellation. Using a significance level of 5\%, test whether the true completion proportion is lower than 64\%.
	
	\subsection*{C4}
	\begin{itemize}
	    \item \textbf{Hypotheses:} \(H_0\colon p=0.64\), \(H_a\colon p<0.64\).
	    \item \textbf{Conditions:} \(np_0=220(0.64)=140.8\ge 10\), \(n(1-p_0)=220(0.36)=79.2\ge 10\).
	    \item \textbf{Sample statistic:} \(\hat p=129/220\approx 0.586\).
	    \item \textbf{Test statistic:}
	    \[
	    z=\frac{0.586-0.64}{\sqrt{\frac{0.64(0.36)}{220}}}\approx -1.66.
	    \]
	    \item \textbf{\(p\)-value:} \(P(Z\le -1.66)\approx 0.0485\), or about 4.85\%.
	    \item \textbf{Decision:} Reject \(H_0\) because 4.85\% is less than 5\%.
	    \item \textbf{Conclusion:} There is sufficient evidence that the completion rate is lower than 64\%.
	\end{itemize}
	
	\section{Average one-way commute time for office staff}
	A company dashboard sets 42 minutes as the benchmark average one-way commute time for staff. A random sample of 49 employees has an average one-way commute of 43.0 minutes, and the population standard deviation is known to be 8 minutes. Using a significance level of 5\%, test whether the true average commute time is higher than 42 minutes.
	
	\subsection*{C4}
	\begin{itemize}
	    \item \textbf{Hypotheses:} \(H_0\colon \mu=42\), \(H_a\colon \mu>42\).
	    \item \textbf{Sample statistic:} \(\bar x=43.0\).
	    \item \textbf{Test statistic:}
	    \[
	    z=\frac{43.0-42}{8/\sqrt{49}}=\frac{1.0}{1.143}\approx 0.88.
	    \]
	    \item \textbf{\(p\)-value:} \(P(Z\ge 0.88)\approx 0.1894\), or about 18.94\%.
	    \item \textbf{Decision:} Fail to reject \(H_0\) because 18.94\% is greater than 5\%.
	    \item \textbf{Conclusion:} There is not enough evidence that the true average commute time is higher than 42 minutes.
	\end{itemize}

	    \item \textbf{Decision at 1\%:} Reject \(H_0\) because 0.51\% is less than 1\%.

	    \item \textbf{\(p\)-value:} \(2P(Z\ge 2.28)\approx 0.0226\), or about 2.26\%.
	    \item \textbf{Decision:} Reject \(H_0\) because 2.26\% is less than 5\%.
	    \item \textbf{Conclusion:} There is sufficient evidence that the true proportion using reminders is different from 52\%.
	\end{itemize}
