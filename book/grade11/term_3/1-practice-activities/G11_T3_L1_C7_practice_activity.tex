\ExamSection{Evaluated criteria}
\begin{ExamCriteria}
    \item C7: Contrasts p-values in hypothesis tests with the reference probability, and justifies decisions in context.
\end{ExamCriteria}

\ExamSection{Problems}
\begin{ExamProblems}

\item
\section{Quality control: defective export units}
A factory tests whether the defective rate is above 3\%. Let \(p\) be the true defective proportion.
Use \(H_0: p\le 0.03\), \(H_1: p>0.03\).
A sample test reports \(p\)-value \(=0.018\) with reference probability \(\alpha=0.05\).
\begin{enumerate}
    \item State the reported \(p\)-value and reference probability.
    \item Decide whether the \(p\)-value is smaller than, equal to, or greater than \(\alpha\).
    \item State the decision about \(H_0\).
    \item Interpret the decision for shipment policy in context.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item Reported \(p\)-value: \(0.018\). Reference probability: \(\alpha=0.05\).
    \item Comparison: \(0.018<0.05\).
    \item Decision: reject \(H_0\).
    \item Contextual interpretation: there is sufficient evidence, at the 5\% reference probability, that the defective rate is above 3\%, so stopping shipment for full inspection is justified.
\end{itemize}

\item
\section{Logistics: on-time delivery target}
A logistics team checks whether on-time delivery has fallen below 92\%. Let \(p\) be the true on-time proportion.
Use \(H_0: p\ge 0.92\), \(H_1: p<0.92\).
A monthly audit reports \(p\)-value \(=0.041\) at reference probability \(\alpha=0.05\).
\begin{enumerate}
    \item Identify the \(p\)-value and the reference probability.
    \item Compare the \(p\)-value with \(\alpha\).
    \item State whether to reject or fail to reject \(H_0\).
    \item Explain what this means for launching a performance-improvement plan.
    \item State one thing this result does \emph{not} prove.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(p\)-value \(=0.041\), reference probability \(\alpha=0.05\).
    \item Since \(0.041<0.05\), the \(p\)-value is smaller than the threshold.
    \item Decision: reject \(H_0\).
    \item Interpretation: evidence is strong enough at the 5\% level to conclude on-time delivery is below 92\%, so an improvement plan should be launched.
    \item Limitation: this does not prove every delivery is late or quantify exactly how far below 92\% the true rate is.
\end{itemize}

\item
\section{Port operations: mean customs processing time}
Port managers test whether average customs processing time exceeds 40 minutes. Let \(\mu\) be the true mean.
Use \(H_0: \mu=40\), \(H_1: \mu>40\).
The test output gives \(p\)-value \(=0.006\) with reference probability \(\alpha=0.05\).
\begin{enumerate}
    \item Compare the \(p\)-value and \(\alpha\).
    \item State the decision about \(H_0\).
    \item Explain why that decision follows from the comparison.
    \item Interpret the decision for staffing policy.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(p\)-value \(=0.006\), \(\alpha=0.05\).
    \item Comparison: \(0.006<0.05\) (clearly below).
    \item Decision: reject \(H_0\).
    \item Reason: because the \(p\)-value is below the reference probability, the sample result is unlikely under \(H_0\), so evidence supports \(H_1\).
    \item Context: managers have enough evidence that mean processing time is above 40 minutes, supporting additional staffing.
\end{itemize}

\item
\section{Macroeconomics: mean inflation monitoring}
Analysts test whether mean monthly inflation is above 4.0\%. Let \(\mu\) be the true mean monthly inflation rate.
Use \(H_0: \mu\le 4.0\), \(H_1: \mu>4.0\).
A report gives \(p\)-value \(=0.27\) with reference probability \(\alpha=0.05\).
\begin{enumerate}
    \item Compare \(0.27\) and \(0.05\).
    \item State the decision about \(H_0\).
    \item Explain why this comparison leads to that decision.
    \item Interpret what policymakers can conclude, and what they cannot conclude.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(p\)-value \(=0.27\), reference probability \(\alpha=0.05\).
    \item Comparison: \(0.27>0.05\) (clearly above).
    \item Decision: fail to reject \(H_0\).
    \item Reason: since the \(p\)-value is larger than the threshold, evidence is not strong enough at 5\% to support \(H_1\).
    \item Context: analysts do not have enough evidence that mean inflation exceeds 4.0\%; this does not prove inflation is exactly 4.0\% or lower forever.
\end{itemize}

\item
\section{Banking: loan approval fairness check}
A regulator tests whether the approval rate for small-business loans is below 60\%.
Let \(p\) be the true approval proportion. Use \(H_0: p\ge 0.60\), \(H_1: p<0.60\).
The test reports \(p\)-value \(=0.049\) and reference probability \(\alpha=0.05\).
\begin{enumerate}
    \item Compare the \(p\)-value with the reference probability.
    \item State the correct decision about \(H_0\).
    \item Interpret the decision for whether a fairness audit should start.
    \item Explain why being close to \(\alpha\) does not change the comparison rule.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(p\)-value \(=0.049\), \(\alpha=0.05\).
    \item Comparison: \(0.049<0.05\), even though it is very close.
    \item Decision: reject \(H_0\).
    \item Context: there is just enough evidence at the 5\% threshold that approval may be below 60\%, so a fairness audit is justified.
    \item Rule reminder: the decision comes from the inequality \(p\)-value \(<\alpha\), not from how large the gap looks.
\end{itemize}

\item
\section{Digital payments: fraud checks in two regions}
A payment platform runs separate fraud-rate tests in Region A and Region B.
For each region, use \(H_0: p\le 0.02\), \(H_1: p>0.02\), where \(p\) is the fraud proportion.
Region A reports \(p\)-value \(=0.012\) with \(\alpha=0.05\).
Region B reports \(p\)-value \(=0.19\) with \(\alpha=0.05\).
\begin{enumerate}
    \item Compare each \(p\)-value with its reference probability.
    \item State the decision for Region A and Region B separately.
    \item Interpret each decision for activating strict blocking rules.
    \item Briefly explain why the two regions can have different decisions under the same \(\alpha\).
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item Region A: \(0.012<0.05\) \(\Rightarrow\) reject \(H_0\).
    \item Region B: \(0.19>0.05\) \(\Rightarrow\) fail to reject \(H_0\).
    \item Context: Region A has sufficient evidence that fraud exceeds 2\%, so strict blocking is supported; Region B does not have sufficient evidence, so automatic strict blocking is not supported yet.
    \item Different sample outcomes can produce different \(p\)-values, so decisions can differ even with the same reference probability.
\end{itemize}

\item
\section{Telecom: churn result under two reference probabilities}
A telecom provider tests whether monthly churn exceeds 12\%. Let \(p\) be true churn proportion.
Use \(H_0: p\le 0.12\), \(H_1: p>0.12\).
The study reports one \(p\)-value: \(0.047\).
Compare this same test result against two reference probabilities: \(\alpha=0.05\) and \(\alpha=0.01\).
\begin{enumerate}
    \item Compare \(0.047\) with \(0.05\), then with \(0.01\).
    \item State the decision at each reference probability.
    \item Explain why the decision changes or does not change when the threshold changes.
    \item Give the practical interpretation for retention discounts at each threshold.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item At \(\alpha=0.05\): \(0.047<0.05\) \(\Rightarrow\) reject \(H_0\).
    \item At \(\alpha=0.01\): \(0.047>0.01\) \(\Rightarrow\) fail to reject \(H_0\).
    \item The decision changes because the same \(p\)-value is below one reference probability but above the stricter one.
    \item Context: at 5\% the evidence supports immediate retention discounts; at 1\% the evidence is not strong enough for that same action.
\end{itemize}

\item
\section{Public services: waiting-time tests across departments}
A city compares waiting-time targets in three departments using independent tests.
Each test uses the reference probability shown below.
\begin{itemize}
    \item Clinic: test if mean wait exceeds 30 minutes; \(p\)-value \(=0.021\), \(\alpha=0.05\).
    \item Licensing office: test if mean wait exceeds 25 minutes; \(p\)-value \(=0.14\), \(\alpha=0.05\).
    \item Call center: test if mean wait exceeds 6 minutes; \(p\)-value \(=0.008\), \(\alpha=0.01\).
\end{itemize}
\begin{enumerate}
    \item For each department, compare the \(p\)-value with its corresponding \(\alpha\).
    \item State reject/fail-to-reject decisions for all three tests.
    \item Interpret each decision in operational terms.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item Clinic: \(0.021<0.05\) \(\Rightarrow\) reject \(H_0\): evidence wait exceeds 30 minutes.
    \item Licensing office: \(0.14>0.05\) \(\Rightarrow\) fail to reject \(H_0\): not enough evidence wait exceeds 25 minutes.
    \item Call center: \(0.008<0.01\) \(\Rightarrow\) reject \(H_0\): evidence wait exceeds 6 minutes even at a stricter threshold.
\end{itemize}

\item
\section{Labor market policy: stability and change across thresholds}
An agency evaluates two independent claims using the same data summary but different reference probabilities.
\begin{itemize}
    \item Claim A (youth unemployment increased): \(p\)-value \(=0.004\).
    \item Claim B (average training completion improved): \(p\)-value \(=0.038\).
\end{itemize}
Use both \(\alpha=0.05\) and \(\alpha=0.01\) for each claim.
\begin{enumerate}
    \item Compare each \(p\)-value with both thresholds.
    \item State the decision for Claim A at 5\% and 1\%.
    \item State the decision for Claim B at 5\% and 1\%.
    \item Identify which claim is stable across thresholds and which changes.
    \item Interpret these patterns in policy language.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item Claim A: \(0.004<0.05\) and \(0.004<0.01\) \(\Rightarrow\) reject \(H_0\) at both levels (stable decision).
    \item Claim B: \(0.038<0.05\) but \(0.038>0.01\) \(\Rightarrow\) reject at 5\%, fail to reject at 1\% (decision changes).
    \item Interpretation: evidence for Claim A remains strong even with a strict reference probability, while Claim B is supported only under the less strict 5\% threshold.
\end{itemize}

\item
\section{Economic monitoring dashboard: mixed evidence across indicators}
A ministry tests four independent indicators and must report decisions at both \(\alpha=0.10\) and \(\alpha=0.05\).
\begin{itemize}
    \item Inflation persistence above target: \(p\)-value \(=0.072\).
    \item Export growth slowdown: \(p\)-value \(=0.049\).
    \item Public transport delays worsening: \(p\)-value \(=0.003\).
    \item Small-business confidence drop: \(p\)-value \(=0.11\).
\end{itemize}
\begin{enumerate}
    \item For each indicator, compare the \(p\)-value with 0.10 and with 0.05.
    \item State reject/fail-to-reject decisions at both thresholds for all indicators.
    \item Identify which indicators pass at 10\% but not at 5\%, which pass at both, and which pass at neither.
    \item Write a short policy summary that correctly reflects what the comparisons do and do not justify.
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item Inflation persistence: \(0.072<0.10\) but \(0.072>0.05\) \(\Rightarrow\) reject at 10\%, fail to reject at 5\%.
    \item Export slowdown: \(0.049<0.10\) and \(0.049<0.05\) \(\Rightarrow\) reject at both thresholds.
    \item Transport delays: \(0.003<0.10\) and \(0.003<0.05\) \(\Rightarrow\) reject at both thresholds.
    \item Confidence drop: \(0.11>0.10\) and \(0.11>0.05\) \(\Rightarrow\) fail to reject at both thresholds.
    \item Policy summary: evidence is strongest for export slowdown and transport delays (robust at both thresholds), moderate for inflation persistence (only at 10\%), and insufficient for a confidence-drop claim at either threshold.
\end{itemize}
