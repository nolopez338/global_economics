\ExamSection{Evaluated criteria}
\begin{ExamCriteria}
    \item C7: Identifies and interprets Type I and Type II errors in context, and evaluates their consequences in hypothesis testing.
\end{ExamCriteria}

\ExamSection{Problems}
\begin{ExamProblems}

\item
\section{Quality control: defective export units}
A factory claims the defective rate is at most 3\%. Let \(p\) be the true defective proportion. Use
\(H_0: p\le 0.03\) and \(H_1: p>0.03\).
If the firm rejects \(H_0\), it will stop shipment and inspect the full batch.
\begin{enumerate}
    \item Identify a Type I error in context.
    \item Identify a Type II error in context.
    \item Which error is more costly? Briefly justify.
    \item If \(\alpha\) is reduced from 0.05 to 0.01, how do Type I and Type II risks change qualitatively?
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(H_0: p\le 0.03\), \(H_1: p>0.03\).
    \item Type I error: concluding the defect rate is above 3\% (and stopping shipment) when the true rate is actually 3\% or less.
    \item Type II error: not stopping shipment when the true defect rate is above 3\%.
    \item More costly error: Type II, because defective products reach customers and can damage contracts and reputation.
    \item Lowering \(\alpha\) decreases Type I risk but increases Type II risk (for fixed sample size).
\end{itemize}

\item
\section{Logistics: on-time delivery target}
A logistics team states at least 92\% of orders arrive on time. Let \(p\) be the on-time proportion.
Use \(H_0: p\ge 0.92\), \(H_1: p<0.92\).
Rejecting \(H_0\) means launching an urgent performance-improvement plan.
\begin{enumerate}
    \item Identify a Type I error in context.
    \item Identify a Type II error in context.
    \item Which error is more costly? Briefly justify.
    \item If sample size \(n\) increases, what happens to Type II risk and power (qualitatively)?
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(H_0: p\ge 0.92\), \(H_1: p<0.92\).
    \item Type I error: concluding on-time delivery is below 92\% when it is actually 92\% or higher.
    \item Type II error: failing to detect that on-time delivery is below 92\% when it truly is below target.
    \item More costly error: Type II, because continuing poor delivery harms customers and future sales.
    \item Increasing \(n\) lowers Type II risk and increases power.
\end{itemize}

\item
\section{Port operations: mean customs processing time}
The benchmark mean processing time is 40 minutes. Let \(\mu\) be the true mean.
Use \(H_0: \mu=40\) and \(H_1: \mu>40\).
Rejecting \(H_0\) means hiring additional customs officers.
A sample test gives \(p\)-value \(=0.032\) at \(\alpha=0.05\).
\begin{enumerate}
    \item State the decision.
    \item Identify a Type I error in context.
    \item Identify a Type II error in context.
    \item Which error is more costly? Briefly justify.
    \item In this decision, which error risk is being directly controlled by choosing \(\alpha\)?
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(H_0: \mu=40\), \(H_1: \mu>40\).
    \item Since \(0.032<0.05\), reject \(H_0\): conclude mean processing time is above 40 minutes.
    \item Type I error: concluding processing is slower than 40 minutes when the true mean is 40 minutes.
    \item Type II error: failing to detect that the true mean exceeds 40 minutes.
    \item More costly error: Type II, because long processing times continue without intervention.
    \item \(\alpha\) directly controls Type I error risk.
\end{itemize}

\section{Banking: loan approval fairness}
A regulator checks whether the approval rate for small-business loans is below 60\%.
Let \(p\) be the true approval proportion. Use \(H_0: p\ge 0.60\), \(H_1: p<0.60\).
Rejecting \(H_0\) triggers a fairness audit and corrective policy.
A test reports \(p\)-value \(=0.11\) at \(\alpha=0.05\).
\begin{enumerate}
    \item State the decision.
    \item Identify a Type I error in context.
    \item Identify a Type II error in context.
    \item Which error is more costly? Briefly justify.
    \item In this setting, what does choosing \(\alpha=0.05\) limit?
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(H_0: p\ge 0.60\), \(H_1: p<0.60\).
    \item Since \(0.11>0.05\), fail to reject \(H_0\).
    \item Type I error: launching a fairness audit when the true approval rate is at least 60\%.
    \item Type II error: missing a genuinely low approval rate (below 60\%).
    \item More costly error: Type II, because unfair access to credit can continue.
    \item \(\alpha=0.05\) limits the probability of a Type I error.
\end{itemize}

Rejecting \(H_0\) means pausing the algorithm and switching to manual review.
\begin{enumerate}

    \item Identify a Type II error in context.

\subsection*{C7}

    \item A larger sample usually lowers Type II risk (higher power).

\section{Labor survey: response bias detection}
A labor survey team tests whether nonresponse bias changes the mean wage estimate.
Let \(\mu\) be the true mean wage gap between respondents and nonrespondents.
Use \(H_0: \mu=0\) and \(H_1: \mu\ne 0\).
Rejecting \(H_0\) means redesigning the survey method before publication.
A study finds \(p\)-value \(=0.08\) at \(\alpha=0.10\).
\begin{enumerate}
    \item State the decision.
    \item Identify a Type I error in context.
    \item Identify a Type II error in context.
    \item Which error is more costly? Briefly justify.
    \item If the same \(p\)-value were tested at \(\alpha=0.05\), how would the decision change?
\end{enumerate}

\subsection*{C7}
\begin{itemize}
    \item \(H_0: \mu=0\), \(H_1: \mu\ne 0\).
    \item Since \(0.08<0.10\), reject \(H_0\): evidence of response bias effect.
    \item Type I error: concluding bias exists when the true mean gap is zero.
    \item Type II error: concluding no bias effect when the true mean gap is nonzero.
    \item More costly error: Type II, because published estimates may mislead policy decisions.
    \item At \(\alpha=0.05\), \(0.08>0.05\), so we would fail to reject \(H_0\).
\end{itemize}

    \item Type II error: not expanding enforcement when true compliance is below 85\%.
    \item More costly error: Type II, because weak compliance can undermine policy goals.
    \item Increasing \(\alpha\) raises Type I risk and lowers Type II risk.
\end{itemize}
