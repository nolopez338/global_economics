	\section*{Evaluated criteria}
	\begin{itemize}
		\item C1: Describe the concept of a statistical hypothesis in context.
		\item C2: Identify the null and alternative hypotheses using correct parameter notation.
		\item C3: Determine whether a test is one tailed or two tailed based on the claim.
	\end{itemize}
	
	\section{General problem}
	A benchmark claim may refer either to a population proportion or to a population mean.
	
	For a proportion setting, the claim states that specific percentage (for example, 62\%) of the population has the characteristic of interest. A sample of number of samples (for example, 200) observations gives number of success observed successes (for example, 112). Use a significance level of significance percentage (for example, 5\%).
	
	For a mean setting, the claim states that the population mean is a benchmark mean value (for example, 75). A sample of number of samples (for example, 36) observations gives a sample mean value (for example, 79). If needed, the population standard deviation is given as a known value (for example, 12). Use a significance level of significance percentage (for example, 1\%).
	
	\textbf{Questions/Tasks.}
	\begin{enumerate}
		\item[(C1)] Identify whether the setting is a proportion claim or a mean claim, and map the benchmark value, sample information, and significance level to the correct symbols.
		\item[(C2)] Write the null and alternative hypotheses for the three directions of the claim.
		\begin{itemize}
			\item[(a)] Write the null hypothesis as ``parameter equals the benchmark'' and the alternative as ``parameter is lower than the benchmark''.
			\item[(b)] Write the null hypothesis as ``parameter equals the benchmark'' and the alternative as ``parameter is higher than the benchmark''.
			\item[(c)] Write the null hypothesis as ``parameter equals the benchmark'' and the alternative as ``parameter is different from the benchmark''.
		\end{itemize}
		\item[(C3)] Classify the tail type for each direction of the claim and justify it in words.
	\end{enumerate}
	
	\subsection*{C1}
	\begin{itemize}
		\item Identify whether the problem is about a population proportion or a population mean.
		\item For a proportion problem:
		\begin{itemize}
			\item state that \(p\) is the true population proportion,
			\item state that \(p_0\) is the benchmark value (for example, 62\%),
			\item define \(n\) as the sample size (for example, 200),
			\item define \(x\) as the observed count (for example, 112),
			\item define \(\alpha\) as the significance level (for example, 5\%).
		\end{itemize}
		\item For a mean problem:
		\begin{itemize}
			\item state that \(\mu\) is the true population mean,
			\item state that \(\mu_0\) is the benchmark value (for example, 75),
			\item define \(n\) as the sample size (for example, 36),
			\item define \(\bar{x}\) as the sample mean (for example, 79),
			\item define \(\sigma\) as the known population standard deviation when it is provided (for example, 12),
			\item define \(\alpha\) as the significance level (for example, 1\%).
		\end{itemize}
		\item Match each symbol correctly to the numerical values given in the statement.
		\item Parts ``lower'', ``higher'', and ``different'' use the same parameter and benchmark, but different alternative claims.
	\end{itemize}
	
	\subsection*{C2}
	\begin{itemize}
		\item Write the hypotheses correctly for each part (a), (b), and (c), and include \(\alpha\) in each case.
		\item For a proportion problem, use \(p\) and \(p_0\):
		\begin{itemize}
			\item (a) \(H_0\colon p=p_0\), \(H_a\colon p<p_0\), with the stated significance level. In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is lower than the benchmark.
			\item (b) \(H_0\colon p=p_0\), \(H_a\colon p>p_0\), with the stated significance level. In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is higher than the benchmark.
			\item (c) \(H_0\colon p=p_0\), \(H_a\colon p\ne p_0\), with the stated significance level. In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is different from the benchmark.
		\end{itemize}
		\item For a mean problem, use \(\mu\) and \(\mu_0\):
		\begin{itemize}
			\item (a) \(H_0\colon \mu=\mu_0\), \(H_a\colon \mu<\mu_0\), with the stated significance level. In words, the null hypothesis says that the population mean equals the benchmark, while the alternative says that the population mean is lower than the benchmark.
			\item (b) \(H_0\colon \mu=\mu_0\), \(H_a\colon \mu>\mu_0\), with the stated significance level. In words, the null hypothesis says that the population mean equals the benchmark, while the alternative says that the population mean is higher than the benchmark.
			\item (c) \(H_0\colon \mu=\mu_0\), \(H_a\colon \mu\ne \mu_0\), with the stated significance level. In words, the null hypothesis says that the population mean equals the benchmark, while the alternative says that the population mean is different from the benchmark.
		\end{itemize}
		\item In every case, keep the null hypothesis as ``parameter equals the benchmark'' and match the inequality direction to lower, higher, or different.
	\end{itemize}
	
	\subsection*{C3}
	\begin{itemize}
		\item Classify the tail type for each direction of the claim and justify it in words.
		\begin{itemize}
			\item (a) left-tailed test, because the claim says ``lower''.\LeftTailGraph
			\item (b) right-tailed test, because the claim says ``higher''.\RightTailGraph
			\item (c) two-tailed test, because the claim says ``different''.\TwoTailGraph
		\end{itemize}
		\item This mapping is the same for both proportion and mean tests.
	\end{itemize}

	\section{Daily app check-ins for students and recent graduates}
		A university career center uses a mobile app to support students and recent graduates with internship alerts, interview reminders, and short productivity check-ins. The center is reviewing whether users are staying engaged enough for the app to justify next semester's funding, especially because many users are balancing classes, part-time work, and job applications. The current benchmark says that 62\% of active users complete the daily check-in. A class project reviews 200 active users and finds that 112 completed the daily check-in. They will use a 5\% significance level for this check.
		
		\textbf{Questions/Tasks.}
		\begin{enumerate}
			\item[(C1)] Identify whether the claim is about a population proportion or a population mean, and match the parameter, the benchmark value, the sample size, and the observed sample result for the daily check-in proportion to the symbols used in the solution.
			\item[(C2)] Write hypotheses for each case using the stated significance level.
			\begin{itemize}
				\item[(a)] Using a significance level of 5\%, write the null and alternative hypotheses for the case where the true daily check-in proportion is lower than 62\%.
				\item[(b)] Using a significance level of 5\%, write the null and alternative hypotheses for the case where the true daily check-in proportion is higher than 62\%.
				\item[(c)] Using a significance level of 5\%, write the null and alternative hypotheses for the case where the true daily check-in proportion is different from 62\%.
			\end{itemize}
			\item[(C3)] For each case, classify the test as left-tailed, right-tailed, or two-tailed and justify using the wording of the alternative claim.
		\end{enumerate}
		
		\subsection*{C1}
		\begin{itemize}
			\item This is a population proportion setting.
			\item Let \(p\) be the true population proportion of active users who check in daily.
			\item Let \(p_0\) be the benchmark proportion, with \(p_0=62\%\).
			\item Let \(n\) be the sample size, with \(n=200\).
			\item Let \(x\) be the observed number of daily check-ins, with \(x=112\).
			\item Let \(\alpha\) be the significance level, with \(\alpha=5\%\).
			\item Parts ``lower'', ``higher'', and ``different'' use the same parameter \(p\) and the same benchmark \(p_0\), but different alternative claims.
		\end{itemize}
		
		\subsection*{C2}
		\textbf{(a)} \(H_0\colon p=0.62,\; H_a\colon p<0.62\), \(\alpha=0.05\). In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is lower than the benchmark.
		
		\textbf{(b)} \(H_0\colon p=0.62,\; H_a\colon p>0.62\), \(\alpha=0.05\). In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is higher than the benchmark.
		
		\textbf{(c)} \(H_0\colon p=0.62,\; H_a\colon p\ne 0.62\), \(\alpha=0.05\). In words, the null hypothesis says that the population proportion equals the benchmark, while the alternative says that the population proportion is different from the benchmark.
