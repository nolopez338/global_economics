\section{Water Storage Level (Uniform)}

\textbf{Problem.}
Let $X$ be the water storage level (in m) with support $[0,4]$ and density
\[
f(x)=k\quad\text{for }0\le x\le 4.
\]

\textbf{Question.}
\begin{itemize}
  \item \textbf{C1 Calculations:} Find $k$ so $f(x)$ is a valid PDF.
  \item \textbf{C1 Interpretation:} Explain what this model says about equal-length intervals.
  \item \textbf{Extra analysis:} Find $P(1\le X\le 3)$.
\end{itemize}
