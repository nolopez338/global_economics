Continuous Probability Distributions

Density-based models for continuous variables, risk, and portfolio evaluation with normal and related distributions.

Key Concepts

  • PDF (f(x)): Probability density; total area under curve is 1.
  • CDF (F(x)): Probability X ≤ x, found by integrating the PDF.
  • Expected value: E[X] = ∫ x·f(x) dx; Variance: Var(X) = ∫ (x−μ)^2 f(x) dx.

Common Distributions

  • Normal(μ, σ²): Bell-shaped; standardize with Z = (X−μ)/σ for table lookup.
  • Exponential(λ): Time between Poisson events; PDF λe^{−λx} for x ≥ 0; memoryless.
  • Uniform(a, b): Constant density 1/(b−a) on [a, b]; mean (a+b)/2.

Example Calculations

Normal tail: If X ~ Normal(50, 9), then Z = (55−50)/3 ≈ 1.67. Using a Z-table, P(X ≥ 55) ≈ 0.0475.
Exponential probability: If service times are Exponential(λ=0.5), then P(X ≤ 2) = 1 − e^{−0.5·2} ≈ 0.632.
