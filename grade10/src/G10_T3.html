  • PDF (f(x)): Probability density; total area under curve is 1.
  • CDF (F(x)): Probability X ≤ x, found by integrating the PDF.
  • Expected value: E[X] = ∫ x·f(x) dx; Variance: Var(X) = ∫ (x−μ)^2 f(x) dx.
  • Normal(μ, σ²): Bell-shaped; standardize with Z = (X−μ)/σ for table lookup.
  • Exponential(λ): Time between Poisson events; PDF λe^{−λx} for x ≥ 0; memoryless.
  • Uniform(a, b): Constant density 1/(b−a) on [a, b]; mean (a+b)/2.

This practice page includes worked discrete and continuous probability-function problems focused on validating distributions, computing event probabilities, finding density constants, and interpreting results in context without integration.

Go to C1 Practice activity.

  • Problem 1 — App Notifications per Hour: Discrete pmf validity and cumulative probability \(P(X\le 2)\).
  • Problem 2 — Defective Items in a Batch of 3: Discrete pmf validity and interval probability \(P(1\le X\le 2)\).
  • Problem 3 — Late Deliveries in One Day: Discrete pmf validity and tail probability \(P(X\ge 2)\).
  • Problem 4 — Customers in a 10-Minute Window: Discrete pmf validity and complement probability \(P(X\ne 4)\).
  • Problem 5 — Correct Answers on a Short Quiz: Discrete pmf validity and upper-tail probability \(P(X\ge 3)\).
  • Problem 6 — Water Tank Fill Level: Constant density on \([0,5]\), solve for \(k\), then compute \(P(1\le X\le 4)\).
  • Problem 7 — Ramp Travel Time: Linear density on \([2,8]\), solve for \(k\), then compute \(P(T>6)\).
  • Problem 8 — Air Quality Index Block Model: Piecewise-constant density, solve for \(k\), then compute \(P(0.5\le A\le 2.5)\).
  • Problem 9 — Symmetric Delivery Window: Triangular piecewise density, solve for \(k\), then compute \(P(0\le Y\le 2)\).
  • Problem 10 — Machine Vibration Level: Two-block density, solve for \(k\), then compute \(P(V\ge 3)\).
  • Problem 11 — Straight-Line Cooling Error: Decreasing linear density, solve for \(k\), then compute \(P(2\le E\le 4)\).
  • Problem 12 — Traffic Gap with Flat Peak: Piecewise triangle-rectangle-triangle density, solve for \(k\), then compute \(P(1\le G\le 5)\).
  • Problem 13 — Quality Score Trapezoid: Linear trapezoid density, solve for \(k\), then compute \(P(4\le Q\le 7)\).
  • Problem 14 — Centered Error Triangle: Symmetric triangular density, solve for \(k\), then compute \(P(-1\le Z\le 1)\).
  • Problem 15 — Three-Level Service Time: Three-piece constant density, solve for \(k\), then compute \(P(S<7)\).

This practice page includes worked continuous probability-density problems focused on linear and piecewise-linear behavior, geometric PDF validation, and probability computation by area (without integration).

Go to C3 Practice activity.

  • Problem 1 — Introductory Linear Density: Identify support, validate a linear PDF geometrically, and compute \(P(0\le X\le 3)\).
  • Problem 2 — Basic Symmetric Triangular Density: Validate a symmetric triangular density and compute \(P(2\le Y\le 6)\) using complements.
  • Problem 3 — Basic Asymmetric Triangular Density: Validate an asymmetric triangular density and compute \(P(1\le Z\le 7)\) by subtracting side areas.
  • Problem 4 — Triangular Density with Split-Interval Areas: Compute \(P(2\le T\le 5)\) by splitting at the peak and compare symmetric interval probabilities.
  • Problem 5 — Richer Peaked Piecewise Linear Shape: Validate a peaked piecewise-linear density and compare interval probabilities from shaded graph areas.
