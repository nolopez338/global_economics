Narrative problem description

A firm is considering launching a project whose profit depends on whether state \(S_1\) (probability \(p\)) or state \(S_2\) (probability \(1-p\)) occurs, with payoffs listed in the table; the task is to use expected value to determine the values of \(p\) for which the project is profitable.

Summary description

  • Economic context and transaction: a firm is evaluating whether to launch a single project based on expected profit.
  • Decision alternatives: undertake the project (single alternative evaluated with expected value).
  • States of nature: \(S_1\) and \(S_2\).
  • State probabilities: \(P(S_1)=p\), \(P(S_2)=1-p\).
  • Payoffs defined as: project profit is 18 in \(S_1\) and -4 in \(S_2\), as shown in the table.

Payoff table:
S1
\((p)\)		S2
\((1-p)\)
Project18-4
↑

Structured solution

\[ EV = 18p + (-4)(1-p) = 22p - 4 \] Profitability condition: \[ EV>0 \Rightarrow 22p-4>0 \]

Payoff table:
S1
\((p)\)		S2
\((1-p)\)
A300

C1412

Let \(p\) be the probability of state \(S_1\). Determine for which values of \(p\) each plan is optimal.
↑

Structured solution

\[ EV(A)=30p,\quad EV(B)=20p+8(1-p)=12p+8,\quad EV(C)=14p+12(1-p)=2p+12 \] Pairwise comparison conditions in \(>0\) form: \[ EV(A)-EV(B)>0 \Rightarrow 30p-(12p+8)>0 \Rightarrow 18p-8>0 \] \[ EV(B)-EV(C)>0 \Rightarrow (12p+8)-(2p+12)>0 \Rightarrow 10p-4>0 \] \[ EV(A)-EV(C)>0 \Rightarrow 30p-(2p+12)>0 \Rightarrow 28p-12>0 \]
