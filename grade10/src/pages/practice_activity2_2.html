A regional solar farm must choose a maintenance plan for the next year. Payoffs are profits (thousand USD).

Alternatives:
• Proactive maintenance
• Minimal maintenance

States of nature and probabilities:
• Sunny year \(P = 0.35\)
• Typical year \(P = 0.45\)
• Cloudy year \(P = 0.20\)

Payoff table (profit in thousand USD):
AlternativeSunny \((0.35)\)Typical \((0.45)\)Cloudy \((0.20)\)
Proactive\(26\)\(20\)\(14\)
Minimal\(30\)\(16\)\(4\)

Question: Use Expected Value only to choose the better plan.
↑

Structured solution

Outcomes and probabilities:
StateProbabilityProactiveMinimal
Sunny\(0.35\)\(26\)\(30\)
Typical\(0.45\)\(20\)\(16\)
Cloudy\(0.20\)\(14\)\(4\)
Expected Value calculations (thousand USD): \[ \begin{aligned} EV(\text{Proactive}) &= 0.35(26) + 0.45(20) + 0.20(14) \\ &= 9.1 + 9 + 2.8 = 20.9 \\ EV(\text{Minimal}) &= 0.35(30) + 0.45(16) + 0.20(4) \\ &= 10.5 + 7.2 + 0.8 = 18.5 \end{aligned} \] Decision: Choose proactive maintenance because its Expected Value \((20.9)\) exceeds \((18.5)\).

AlternativeHigh \((0.40)\)Medium \((0.35)\)Low \((0.25)\)
Fixed lease\(15\)\(12\)\(9\)
Flexible lease\(20\)\(10\)\(-2\)

Question: Use Expected Value only to select the better contract.
↑

Structured solution

Outcomes and probabilities:
StateProbabilityFixed leaseFlexible lease
High occupancy\(0.40\)\(15\)\(20\)

Question: Use Expected Value only to decide which option is better.
↑

Structured solution

Outcomes and probabilities:
StateProbabilityBondVenture

A community group sells raffle tickets for \(\$5\) each. The prizes are:

• \(\$50\) with probability \(0.05\)
• \(\$10\) with probability \(0.20\)
• \(\$0\) with probability \(0.75\)

The ticket fee is paid no matter what. Payoffs below are net gains (prize − fee).

Question: Use Expected Value only to decide whether buying a ticket is a good choice.
↑

Structured solution

Outcomes and probabilities (net payoffs):
OutcomeProbabilityNet payoff (USD)
Win \(\$50\)\(0.05\)\(50 - 5 = 45\)
Win \(\$10\)\(0.20\)\(10 - 5 = 5\)
No prize\(0.75\)\(0 - 5 = -5\)
Expected Value calculation (USD): \[ \begin{aligned} EV &= 0.05(45) + 0.20(5) + 0.75(-5) \\ &= 2.25 + 1 - 3.75 = -0.50 \end{aligned} \] Decision: Do not buy the ticket because the Expected Value is negative \((-0.50)\) per ticket.

A casino offers a prize wheel spin for \(\$8\). The wheel has three outcomes:

• \(\$40\) prize with probability \(0.10\)
• \(\$12\) prize with probability \(0.30\)
• \(\$0\) prize with probability \(0.60\)

Payoffs below are net gains (prize − fee).

Question: Use Expected Value only to decide whether the game is favorable for a player.
↑

Structured solution

Outcomes and probabilities (net payoffs):
OutcomeProbabilityNet payoff (USD)
Win \(\$40\)\(0.10\)\(40 - 8 = 32\)
Win \(\$12\)\(0.30\)\(12 - 8 = 4\)
No prize\(0.60\)\(0 - 8 = -8\)
Expected Value calculation (USD): \[ \begin{aligned} EV &= 0.10(32) + 0.30(4) + 0.60(-8) \\ &= 3.2 + 1.2 - 4.8 = -0.4 \end{aligned} \] Decision: The game is not favorable because the Expected Value is negative \((-0.4)\) per spin.
