Continuous Probability Distributions

Introduction

A continuous random variable can take any value in an interval (or across all real numbers). Instead of listing probabilities at single points, we use a probability density function (PDF).

For a valid PDF, two key properties are required: f(x) ≥ 0 for all x, and the total area under the curve is 1. For continuous variables, probability at a single point is 0, and probabilities on intervals are found using the area under the curve over that interval.

Supported functions and syntax
  • Operators: +, -, *, /, ^, parentheses
  • Variable: x
  • Constants: pi, e
  • Functions: exp, sqrt, abs, ln or log, sin, cos, tan, pow, min, max
  • Examples: 0.5*x, exp(-x), 1/sqrt(2*pi)*exp(-0.5*x^2), (3/8)*(1+x)
