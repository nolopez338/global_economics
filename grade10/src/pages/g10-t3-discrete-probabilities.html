Discrete Probability Distributions (PMF Explorer)

Introduction

A discrete random variable takes separate countable values (for example 0, 1, 2, 3). A probability mass function (PMF) tells us how likely each value is.

For a valid PMF, two key properties must hold:

  • Every probability is non-negative: p(x) ≥ 0.
  • The probabilities add to 1: ∑ p(x) = 1.
Example: Suppose X is the number of correct answers out of 3 guesses. One possible PMF is: p(0)=0.1, p(1)=0.3, p(2)=0.4, p(3)=0.2.
