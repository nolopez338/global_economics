Narrative problem description

A community bookstore must choose between expanding reading space or launching a mobile book cart, while foot traffic can be high (0.55) or low (0.45); profits in thousands of USD are listed in the payoff table, and the task is to select the alternative with minimax regret and compare it to maximax, maximin, and expected value choices.

Summary description

  • Economic context and transaction: a community bookstore is deciding how to update its storefront using projected profits.
  • Decision alternatives: expand reading space; launch a mobile book cart.
  • States of nature: high foot traffic, low foot traffic.
  • State probabilities: \(P(\text{high})=0.55\), \(P(\text{low})=0.45\).
  • Payoffs defined as: profits in thousands of USD, shown in the payoff table for each alternative and traffic state.

Payoff table (given):
AlternativeHigh traffic
\((0.55)\)		Low traffic
\((0.45)\)
Expand space\(48\)\(12\)
Mobile cart\(32\)\(20\)

Question: Use the maximum opportunity (minimax regret) criterion to choose the best alternative. Then compare the choice to Maximax, Maximin, and Expected Value.
↑

Structured solution

Best payoff in each state: \[ \begin{aligned} \text{High traffic: } &\max\{48, 32\} = 48 \\ \text{Low traffic: } &\max\{12, 20\} = 20 \end{aligned} \] Regret table (best payoff − payoff):
AlternativeHigh traffic
\((0.55)\)		Low traffic
\((0.45)\)		Maximum regret
Expand space\(48 - 48 = 0\)\(20 - 12 = 8\)\(8\)
Mobile cart\(48 - 32 = 16\)\(20 - 20 = 0\)\(16\)
The minimax regret choice is expand space because it has the smaller maximum regret \((8)\).
↑
  • Maximax: Choose expand space because its best payoff \((48)\) is larger than the mobile cart’s best payoff \((32)\). A risk-seeking decision-maker accepts uncertainty to chase the biggest gain, so this matches the C6 choice.
  • Maximin: Choose mobile cart because its worst payoff \((20)\) is higher than the expand option’s worst payoff \((12)\). A highly cautious decision-maker avoids low outcomes, so this differs from the C6 choice.
  • Expected Value: The Expected Value favors expand space because the higher payoff in the high-traffic state dominates when weighted by the given probabilities. This aligns with the C6 choice because it balances regret with the more likely outcome.

Narrative problem description

A dairy cooperative must choose between an automated or manual packing line while order size can be high (0.30), medium (0.45), or low (0.25); profits in thousand USD are shown in the payoff table, and the task is to select using the maximum opportunity criterion and compare with maximax, maximin, and expected value.

Summary description

  • Economic context and transaction: a dairy cooperative is selecting a packing line based on projected profit.
  • Decision alternatives: automated packing line; manual packing line.
  • States of nature: high, medium, or low order size.
  • State probabilities: high 0.30, medium 0.45, low 0.25.
  • Payoffs defined as: profits in thousand USD shown in the payoff table for each alternative and order size.

Payoff table (profit in thousand USD, given):
AlternativeHigh
\((0.30)\)		Medium
\((0.45)\)		Low
\((0.25)\)
Automated\(42\)\(28\)\(8\)
Manual\(30\)\(24\)\(12\)

Question: Apply the maximum opportunity criterion and then compare with Maximax, Maximin, and Expected Value.
↑

Structured solution

Best payoff in each state: \[ \begin{aligned} \text{High: } &\max\{42, 30\} = 42 \\ \text{Medium: } &\max\{28, 24\} = 28 \\ \text{Low: } &\max\{8, 12\} = 12 \end{aligned} \] Regret table (best payoff − payoff):
AlternativeHigh
\((0.30)\)		Medium
\((0.45)\)		Low
\((0.25)\)		Maximum regret
Automated\(42 - 42 = 0\)\(28 - 28 = 0\)\(12 - 8 = 4\)\(4\)
Manual\(42 - 30 = 12\)\(28 - 24 = 4\)\(12 - 12 = 0\)\(12\)
The minimax regret choice is automated packing because its maximum regret is \(4\), which is smaller than \(12\).
↑
  • Maximax: Choose automated packing because its best payoff \((42)\) exceeds the manual line’s best payoff \((30)\). This matches the C6 decision for a risk-seeking manager focused on the highest possible gain.
  • Maximin: Choose manual packing because its worst payoff \((12)\) is higher than the automated line’s worst payoff \((8)\). A risk-averse manager prioritizes the safest minimum, so this differs from the C6 choice.
  • Expected Value: The Expected Value favors automated packing because the high and medium outcomes carry more total probability and higher payoffs. This aligns with the C6 decision.

Narrative problem description

A cooperative exporting handmade crafts must choose whether to ship by air, by sea, or delay shipment; demand and exchange rates form states \(S1\)–\(S4\) with probabilities 0.20, 0.30, 0.25, and 0.25, and profits in thousand USD for each shipping option are listed in the payoff table for use with minimax regret, maximax, maximin, and expected value.

Summary description

  • Economic context and transaction: a cooperative selects a shipping plan for exported handmade crafts based on profit outcomes.
  • Decision alternatives: ship by air, ship by sea, or delay shipment.
  • States of nature: S1 high demand + strong currency, S2 high demand + weak currency, S3 low demand + strong currency, S4 low demand + weak currency.
  • State probabilities: \(P(S1)=0.20\), \(P(S2)=0.30\), \(P(S3)=0.25\), \(P(S4)=0.25\).
  • Payoffs defined as: profits in thousand USD listed in the payoff table for each shipping option and state.

Payoff table (profit in thousand USD, given):
AlternativeS1 \((0.20)\)S2 \((0.30)\)S3 \((0.25)\)S4 \((0.25)\)
Air\(60\)\(48\)\(20\)\(10\)
Sea\(50\)\(44\)\(26\)\(18\)
Delay\(35\)\(32\)\(28\)\(24\)

Question: Apply the maximum opportunity criterion and then compare with Maximax, Maximin, and Expected Value.
↑

Structured solution

Best payoff in each state: \[ \begin{aligned} \text{S1: } &\max\{60, 50, 35\} = 60 \\ \text{S2: } &\max\{48, 44, 32\} = 48 \\ \text{S3: } &\max\{20, 26, 28\} = 28 \\ \text{S4: } &\max\{10, 18, 24\} = 24 \end{aligned} \] Regret table (best payoff − payoff):
AlternativeS1S2S3S4Maximum regret
Air\(60 - 60 = 0\)\(48 - 48 = 0\)\(28 - 20 = 8\)\(24 - 10 = 14\)\(14\)
Sea\(60 - 50 = 10\)\(48 - 44 = 4\)\(28 - 26 = 2\)\(24 - 18 = 6\)\(10\)
Delay\(60 - 35 = 25\)\(48 - 32 = 16\)\(28 - 28 = 0\)\(24 - 24 = 0\)\(25\)
The minimax regret choice is ship by sea because it has the smallest maximum regret \((10)\).
↑
  • Maximax: Choose air because its best payoff \((60)\) is the largest of all options. This differs from the C6 choice because it ignores regret and focuses only on the single best case.
  • Maximin: Choose delay because its worst payoff \((24)\) is higher than the worst outcomes for air \((10)\) or sea \((18)\). A very cautious decision-maker would accept slower delivery to protect against bad outcomes, so this differs from the C6 choice.
  • Expected Value: The Expected Value favors sea because it balances strong payoffs with moderate risk across all four states. This matches the C6 decision because both approaches reward consistent performance.

Narrative problem description

A student council must choose between smart vending kiosks and pop-up snack stalls for campus events, while campus traffic can be high (0.40), medium (0.35), or low (0.25); profits in thousand USD are given in the payoff table, and the choice should be evaluated with minimax regret and compared to maximax, maximin, and expected value.

Summary description

  • Economic context and transaction: a student council is selecting a vending option for campus events based on projected profit.
  • Decision alternatives: smart vending kiosks; pop-up snack stalls.
  • States of nature: high, medium, or low campus traffic.
  • State probabilities: high 0.40, medium 0.35, low 0.25.
  • Payoffs defined as: profits in thousand USD shown in the payoff table for each vending option and traffic level.

Payoff table (profit in thousand USD, given):
AlternativeHigh
\((0.40)\)		Medium
\((0.35)\)		Low
\((0.25)\)
Kiosks\(25\)\(12\)\(-1\)
Pop-up stalls\(23\)\(14\)\(5\)

Question: Apply the maximum opportunity criterion and then compare with Maximax, Maximin, and Expected Value.
↑

Structured solution

Best payoff in each state: \[ \begin{aligned} \text{High: } &\max\{25, 23\} = 25 \\ \text{Medium: } &\max\{12, 14\} = 14 \\ \text{Low: } &\max\{-1, 5\} = 5 \end{aligned} \] Regret table (best payoff − payoff):
AlternativeHigh
\((0.40)\)		Medium
\((0.35)\)		Low
\((0.25)\)		Maximum regret
Kiosks\(25 - 25 = 0\)\(14 - 12 = 2\)\(5 - (-1) = 6\)\(6\)
Pop-up stalls\(25 - 23 = 2\)\(14 - 14 = 0\)\(5 - 5 = 0\)\(2\)
The minimax regret choice is pop-up stalls because the maximum regret is \(2\), which is lower than \(6\).
↑
  • Maximax: Choose kiosks because its best payoff \((25)\) is larger than the pop-up stalls’ best payoff \((23)\). A risk-seeking decision-maker accepts the chance of losses to chase the largest possible profit, so this differs from the C6 choice.
  • Maximin: Choose pop-up stalls because its worst payoff \((5)\) exceeds the kiosk’s worst payoff \((-1)\). A risk-averse decision-maker avoids negative results, so this matches the C6 choice.
  • Expected Value: The Expected Value favors pop-up stalls because the higher payoffs in medium and low traffic are weighted by substantial probabilities. This aligns with the C6 decision.
