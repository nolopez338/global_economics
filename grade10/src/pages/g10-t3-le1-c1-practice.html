Problem. A continuous random variable is modeled on the interval shown in each case.

Distribution A

\[f(x)=0.25\quad\text{for }0\le x\le 4\]

Distribution B

\[f(x)=0.40\quad\text{for }0\le x\le 4\]

Questions/Tasks.

  • C1 Calculations: Decide which distribution is a valid PDF and which one is not.
  • C1 Interpretation: Justify your answer using the total area under each graph.
  • Extra analysis: For the valid distribution, find \(P(1\le X\le 3)\).
↑

C1 calculations. For a valid PDF, \(f(x)\ge 0\) on its support and total area is \(1\).

  • Distribution A: support \([0,4]\), constant height \(0.25\).
    \[\text{Area}=4(0.25)=1,\]

    so A is valid.

  • Distribution B: support \([0,4]\), constant height \(0.40\).
    \[\text{Area}=4(0.40)=1.6\ne1,\]

    so B is not valid.

C2 interpretation.

The valid PDF is uniform on \([0,4]\): it is constant, nonnegative, and gives equal probability to equal-width intervals.

Extra analysis. For the valid distribution A,

\[P(1\le X\le 3)=(3-1)(0.25)=0.5.\]

Final answer: A is valid, B is not valid, and \(P(1\le X\le 3)=0.5\).

Problem. A continuous random variable is modeled on the interval shown in each case.

Distribution A

\[f(x)=\frac{x}{8}\quad\text{for }0\le x\le 4\]

Distribution B

\[f(x)=\frac{x}{6}\quad\text{for }0\le x\le 4\]

Questions/Tasks.

  • C1 Calculations: Decide which distribution is a valid PDF and which one is not.
  • C1 Interpretation: Justify your answer using geometric reasoning about the area under each line.
  • Extra analysis: For the valid distribution, find \(P(2\le X\le 4)\).
↑

C1 calculations. Check nonnegativity and total area on support \([0,4]\).

  • Distribution A: \(f(x)=\frac{x}{8}\) forms a triangle with base \(4\) and height \(\frac{4}{8}=\frac12\).
    \[\text{Area}=\frac12(4)\left(\frac12\right)=1,\]

    so A is valid.

  • Distribution B: \(f(x)=\frac{x}{6}\) has base \(4\) and height \(\frac{4}{6}=\frac23\).
    \[\text{Area}=\frac12(4)\left(\frac23\right)=\frac43\ne1,\]

    so B is not valid.

C2 interpretation.

The valid PDF increases linearly on \([0,4]\), so larger \(x\) values have higher density than smaller ones.

Extra analysis. For A,

\[P(2\le X\le4)=1-P(0\le X\le2).\]

On \([0,2]\), area is a triangle with base \(2\) and height \(\frac{2}{8}=\frac14\):

\[P(0\le X\le2)=\frac12(2)\left(\frac14\right)=\frac14.\]

Hence

\[P(2\le X\le4)=1-\frac14=\frac34.\]

Final answer: A is valid, B is not valid, and \(P(2\le X\le4)=\frac34\).

Problem. A continuous random variable is modeled on the interval shown in each case.

Distribution A

\[f(x)=\frac{6-x}{12}\quad\text{for }1\le x\le 5\]

Distribution B

\[f(x)=\frac{6-x}{10}\quad\text{for }1\le x\le 5\]

Questions/Tasks.

  • C1 Calculations: Decide which distribution is a valid PDF and which one is not.
  • C1 Interpretation: Justify your answer using geometric reasoning about the area under each line.
  • Extra analysis: For the valid distribution, find \(P(1\le X\le 3)\).
↑

C1 calculations. On support \([1,5]\), both densities are nonnegative. Check total area:

  • Distribution A: \(f(x)=\frac{6-x}{12}\) gives a trapezium with base \(4\), left height \(f(1)=\frac{5}{12}\), and right height \(f(5)=\frac{1}{12}\).
    \[\text{Area}=\frac{\frac{5}{12}+\frac{1}{12}}{2}\cdot 4=1,\]

    so A is valid.

  • Distribution B: \(f(x)=\frac{6-x}{10}\) gives base \(4\), left height \(f(1)=\frac{5}{10}=\frac12\), and right height \(f(5)=\frac{1}{10}\).
    \[\text{Area}=\frac{\frac12+\frac{1}{10}}{2}\cdot 4=\frac65\ne1,\]

    so B is not valid.

C2 interpretation.

The valid PDF is decreasing on \([1,5]\), so values near \(1\) are more likely than values near \(5\), but values near \(5\) still have positive density.

Extra analysis. For valid distribution A,

\[P(1\le X\le3)=\text{area of trapezium on }[1,3].\]

With heights \(f(1)=\frac{5}{12}\) and \(f(3)=\frac14\), width \(2\):

\[P(1\le X\le3)=\frac{\frac{5}{12}+\frac14}{2}\cdot 2=\frac23.\]

Final answer: A is valid, B is not valid, and \(P(1\le X\le3)=\frac23\).

Problem. Let \(T\) be a startup delay (in s) with support \([0,6]\) and density

\[f(t)=kt\quad\text{for }0\le t\le 6.\]

Questions/Tasks.

  • C1 Calculations: Find \(k\) so \(f(t)\) is a valid PDF.
  • C1 Interpretation: Describe how probability changes as \(t\) increases.
  • Extra analysis: Find \(P(T\ge 4)\).
↑

C1 calculations. For \(f(t)=kt\) on \([0,6]\), area is a triangle with base \(6\) and height \(6k\):

\[\frac12(6)(6k)=18k=1\implies k=\frac1{18}.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. For \(P(T\ge4)\), use area from \(4\) to \(6\) (a trapezium): heights are \(4k\) and \(6k\), width \(2\).

\[P(T\ge4)=\frac{4k+6k}{2}\cdot2=10k=\frac{10}{18}=\frac59.\]

Final answer: \(k=\frac1{18}\) and \(P(T\ge4)=\frac59\).

Problem. Let \(Q\) be a packing score with support \([2,8]\) and density

\[f(q)=k(q-1)\quad\text{for }2\le q\le 8.\]

Questions/Tasks.

  • C1 Calculations: Find \(k\) so \(f(q)\) is a valid PDF.
  • C1 Interpretation: Compare density near \(q=3\) and \(q=7\).
  • Extra analysis: Find \(P(4\le Q\le 6)\).
↑

C1 calculations. On \([2,8]\), \(f(q)=k(q-1)\) is linear and positive. Total area is a trapezium with width \(6\), heights

\[f(2)=k,\quad f(8)=7k.\]

So

\[\frac{k+7k}{2}\cdot6=24k=1\implies k=\frac1{24}.\]

Compare densities:

\[f(3)=2k,\quad f(7)=6k,\]

so density near \(q=7\) is three times density near \(q=3\).

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. For

\[P(4\le Q\le6)=\text{trapezium area on }[4,6],\]

with heights \(f(4)=3k\), \(f(6)=5k\), width \(2\):

\[P(4\le Q\le6)=\frac{3k+5k}{2}\cdot2=8k=\frac13.\]

Final answer: \(k=\frac1{24}\) and \(P(4\le Q\le6)=\frac13\).

Problem. Let \(M\) be rainfall amount (in mm) with support \([3,b]\) and density

\[f(m)=\frac16\quad\text{for }3\le m\le b.\]

Questions/Tasks.

  • C1 Calculations: Find \(b\) so \(f(m)\) is a valid PDF.
  • C1 Interpretation: State two different intervals with equal probability and justify.
  • Extra analysis: Find \(P(4.5\le M\le 7.5)\).
↑

C1 calculations. The height is fixed at \(\frac16\), so the total area must be

\[(b-3)\cdot \frac16=1.\]

Hence

\[b-3=6\implies b=9.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. Two equal-length intervals have equal probability, for example:

\[P(3\le M\le5)=P(7\le M\le9)=2\cdot\frac16=\frac13.\]
\[P(4.5\le M\le7.5)=(7.5-4.5)\cdot\frac16=\frac{3}{6}=\frac12.\]

Final answer: \(b=9\), equal-length intervals such as \([3,5]\) and \([7,9]\) have equal probability, and \(P(4.5\le M\le7.5)=\frac12\).

Problem. Let \(B\) be battery discharge measure with support \([0,c]\) and density

\[f(b)=\frac{1}{16}+\frac{b}{8c}\quad\text{for }0\le b\le c.\]

Questions/Tasks.

  • C1 Calculations: Find \(c\) so \(f(b)\) is a valid PDF.
  • C1 Interpretation: Compare likelihood of values in \([0,2]\) and \([6,8]\).
  • Extra analysis: Find \(P(2\le B\le 6)\).
↑

C1 calculations. The endpoint values are fixed:

\[f(0)=\frac{1}{16},\quad f(c)=\frac{1}{16}+\frac{c}{8c}=\frac{3}{16}.\]

Total area must be \(1\). Using the trapezium formula,

\[\frac{\frac{1}{16}+\frac{3}{16}}{2}\cdot c=1.\]

So

\[\frac{4/16}{2}\cdot c=1\]
\[\frac{1}{8}c=1\implies c=8.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. Using \(c=8\), the density is

\[f(b)=\frac{1}{16}+\frac{b}{64}\quad\text{for }0\le b\le 8.\]

For the comparison:

\[P(0\le B\le2)=\frac{\frac{1}{16}+\frac{3}{32}}{2}\cdot2=\frac{5}{32},\qquad P(6\le B\le8)=\frac{\frac{5}{32}+\frac{3}{16}}{2}\cdot2=\frac{11}{32},\]

so \([6,8]\) is more likely.

For the requested probability,

\[P(2\le B\le6)=\frac{f(2)+f(6)}{2}\cdot4 =\frac{\frac{3}{32}+\frac{5}{32}}{2}\cdot4 =\frac12.\]

Final answer: \(c=8\) and \(P(2\le B\le6)=\frac12\).

Problem. Let \(D\) be a delivery score with support \([-3,c]\) and density

\[f(d)=\frac{1}{40}+\frac{d+3}{5(c+3)}\quad\text{for }-3\le d\le c.\]

Questions/Tasks.

  • C1 Calculations: Find \(c\) so \(f(d)\) is a valid PDF.
  • C1 Interpretation: Explain why the model is nonnegative on its support.
  • Extra analysis: Find \(P(-1\le D\le 3)\).
↑

C1 calculations. The endpoint values are fixed:

\[f(-3)=\frac{1}{40},\quad f(c)=\frac{1}{40}+\frac{c+3}{5(c+3)}=\frac{1}{40}+\frac15=\frac{9}{40}.\]

Since the graph is a line joining two positive endpoint values, the density is nonnegative on its support.

The total area must be \(1\). Using the trapezium formula,

\[\frac{\frac{1}{40}+\frac{9}{40}}{2}\cdot(c+3)=1.\]

So

\[\frac{10/40}{2}\cdot(c+3)=1\]
\[\frac{1}{8}(c+3)=1\implies c+3=8\implies c=5.\]

Now substitute this value into the density:

\[f(d)=\frac{1}{40}+\frac{d+3}{40}=\frac{d+4}{40}\quad\text{for }-3\le d\le 5.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. For the requested probability,

\[P(-1\le D\le3)=\frac{f(-1)+f(3)}{2}\cdot4.\]

With

\[f(-1)=\frac{3}{40},\quad f(3)=\frac{7}{40},\]

we get

\[P(-1\le D\le3)=\frac{\frac{3}{40}+\frac{7}{40}}{2}\cdot4=\frac12.\]

Final answer: \(c=5\), the density is nonnegative since both endpoint heights are positive and the graph is linear on the support, and \(P(-1\le D\le3)=\frac12\).

Problem. Let \(W\) be clinic waiting time (in min) with support \([2,c]\). The density is a line on this support, with endpoint values

\[f(2)=\frac18 \quad\text{and}\quad f(c)=\frac18.\]

Questions/Tasks.

  • C1 Calculations: Find \(c\) so the model is a valid PDF.
  • C1 Interpretation: Explain how to compare probabilities of two equal-width waiting-time intervals.
  • Extra analysis: Find \(P(W>7)\).
↑

C1 calculations. Since the density is a line and has the same value at both endpoints, it is constant on the support. So the graph is a rectangle of height \(\frac18\) and width \(c-2\).

For a valid PDF, the total area must be \(1\):

\[(c-2)\cdot\frac18=1.\]

Thus

\[c-2=8\implies c=10.\]

For a constant density, equal-width intervals have equal probability because probability is width \(\times \frac18\).

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis.

\[P(W>7)=P(7<W\le10)=(10-7)\cdot\frac18=\frac38.\]

Final answer: \(c=10\) and \(P(W>7)=\frac38\).

Problem. Let \(C\) be a completion fraction with support \([0,b]\). The density is a line on this support, with endpoint values

\[f(0)=1 \quad\text{and}\quad f(b)=0.\]

Questions/Tasks.

  • C1 Calculations: Find \(b\) so the model is a valid PDF.
  • C1 Interpretation: Explain the practical meaning of higher density near \(c=0\) than near the right endpoint.
  • Extra analysis: Find \(P\left(\frac{1}{2}\le C\le \frac{3}{2}\right)\).
↑

C1 calculations. Since the density is a line with endpoint values \(f(0)=1\) and \(f(b)=0\), the graph is a decreasing triangle with base \(b\) and height \(1\).

For a valid PDF, the total area must be \(1\):

\[\frac12(b)(1)=1.\]

Thus

\[b=2.\]

So the density is the line joining \((0,1)\) and \((2,0)\), that is,

\[f(c)=1-\frac{c}{2}\quad\text{for }0\le c\le 2.\]

Higher density near \(c=0\) means lower completion fractions occur more often than values near full completion.

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. For

\[\frac12\le C\le\frac32,\]

use trapezium area. The endpoint heights are

\[f\!\left(\frac12\right)=1-\frac14=\frac34, \qquad f\!\left(\frac32\right)=1-\frac34=\frac14,\]

with width \(1\). So

\[P\!\left(\frac12\le C\le\frac32\right) = \frac{\frac34+\frac14}{2}\cdot1 = \frac12.\]

Final answer: \(b=2\) and \(P\left(\frac12\le C\le\frac32\right)=\frac12\).

Problem. Let \(P\) be a performance measure with support \([1,c]\). The density is a line on this support, with endpoint values

\[f(1)=\frac{9}{40} \quad\text{and}\quad f(c)=\frac{1}{40}.\]

Questions/Tasks.

  • C1 Calculations: Find \(c\) so the model is a valid PDF.
  • C1 Interpretation: Compare \(P(1\le P\le 3)\) and \(P(7\le P\le 9)\) without integration, using geometric reasoning.
  • Extra analysis: Find \(P(3\le P\le 8)\).
↑

C1 calculations. Since the density is a line with endpoint values

\[f(1)=\frac{9}{40},\quad f(c)=\frac{1}{40},\]

the graph is a decreasing trapezium on \([1,c]\).

For a valid PDF, the total area must be \(1\):

\[\frac{\frac{9}{40}+\frac{1}{40}}{2}\cdot(c-1)=1.\]

Thus

\[\frac{10/40}{2}\cdot(c-1)=1\]
\[\frac18(c-1)=1\implies c-1=8\implies c=9.\]

So the density is the line joining \((1,\frac{9}{40})\) and \((9,\frac{1}{40})\), that is,

\[f(p)=\frac{10-p}{40}\quad\text{for }1\le p\le 9.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. Compare intervals geometrically:

\[P(1\le P\le3)=\frac{f(1)+f(3)}{2}\cdot2\]

with

\[f(1)=\frac{9}{40},\quad f(3)=\frac{7}{40}.\]

So

\[P(1\le P\le3)=\frac{\frac{9}{40}+\frac{7}{40}}{2}\cdot2=\frac{16}{40}=\frac25.\]

Also,

\[P(7\le P\le9)=\frac{f(7)+f(9)}{2}\cdot2\]

with

\[f(7)=\frac{3}{40},\quad f(9)=\frac{1}{40}.\]

So

\[P(7\le P\le9)=\frac{\frac{3}{40}+\frac{1}{40}}{2}\cdot2=\frac{4}{40}=\frac1{10}.\]

Therefore,

\[P(1\le P\le3)>P(7\le P\le9).\]

For \(3\le P\le8\):

\[P(3\le P\le8)=\frac{f(3)+f(8)}{2}\cdot5\]

with

\[f(3)=\frac{7}{40},\quad f(8)=\frac{2}{40}.\]

Thus

\[P(3\le P\le8)=\frac{\frac{7}{40}+\frac{2}{40}}{2}\cdot5=\frac{9}{16}.\]

Final answer: \(c=9\), \(P(1\le P\le3)=\frac25>P(7\le P\le9)=\frac1{10}\), and \(P(3\le P\le8)=\frac9{16}\).

Problem. Let \(F\) be a traffic flow index with support \([0,b]\), where \(b>0\), and density

\[f(f)=kf\quad\text{for }0\le f\le b.\]

Questions/Tasks.

  • C1 Calculations: Find \(k\) in terms of \(b\) so \(f(f)\) is a valid PDF.
  • C1 Interpretation: Explain why larger values of \(f\) are more likely than smaller values in equal-width intervals.
  • Extra analysis: Find \(P\left(\frac{b}{2}\le F\le b\right)\).
↑

C1 calculations. For \(f(f)=kf\) on \([0,b]\) (\(b>0\)), area is a triangle with base \(b\) and height \(kb\):

\[\frac12(b)(kb)=\frac{kb^2}{2}=1\implies k=\frac{2}{b^2}.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. For the requested probability,

\[P\!\left(\frac{b}{2}\le F\le b\right)=1-P\!\left(0\le F\le\frac{b}{2}\right).\]

Left part is a triangle with base \(\frac b2\) and height \(k\frac b2\):

\[P\!\left(0\le F\le\frac{b}{2}\right)=\frac12\left(\frac b2\right)\left(k\frac b2\right)=\frac{kb^2}{8}=\frac14.\]

Hence

\[P\!\left(\frac{b}{2}\le F\le b\right)=1-\frac14=\frac34.\]

Final answer: \(k=\frac{2}{b^2}\) and \(P\left(\frac{b}{2}\le F\le b\right)=\frac34\).

Problem. Let \(D\) be a delivery deviation with support \([p,q]\), where \(p<q\), and density

\[f(d)=k(d-p+1)\quad\text{for }p\le d\le q.\]

Questions/Tasks.

  • C1 Calculations: Find \(k\) in terms of \(p\) and \(q\) so \(f(d)\) is a valid PDF.
  • C1 Interpretation: Explain why the density stays positive throughout the support.
  • Extra analysis: Find \(P\left(p+\frac{q-p}{3}\le D\le q\right)\).
↑

C1 calculations. For \(f(d)=k(d-p+1)\) on \([p,q]\) (\(p<q\)):

\[d-p+1\in[1,\,q-p+1],\]

so density is positive on the whole support. Let \(t=q-p>0\). Then heights are \(k\) and \(k(t+1)\), width \(t\), so

\[\frac{k+k(t+1)}{2}\cdot t=\frac{k(t+2)t}{2}=1 \implies k=\frac{2}{(q-p)(q-p+2)}.\]

C2 interpretation.

The PDF graph shows the support and overall shape of the density on that support.

Extra analysis. Now compute \( P\left(p+\frac{q-p}{3}\le D\le q\right) \). Using \(y=d-p\) (\(0\le y\le t\)), this is \(y\in\left[\frac t3,t\right]\) with density \(k(y+1)\):

\[P=\int_{t/3}^{t}k(y+1)\,dy =\frac{k}{2}\left[(y+1)^2\right]_{t/3}^{t} =\frac{k}{2}\left((t+1)^2-\left(\frac t3+1\right)^2\right).\]

Substitute \(k=\frac{2}{t(t+2)}\) and simplify:

\[P=\frac{4(2t+3)}{9(t+2)}=\frac{4(2(q-p)+3)}{9(q-p+2)}.\]

Final answer: \(k=\frac{2}{(q-p)(q-p+2)}\) and

\[P\!\left(p+\frac{q-p}{3}\le D\le q\right)=\frac{4(2(q-p)+3)}{9(q-p+2)}.\]
