	\section{G10\_T1\_L1\_C1-C2\_practice\_activity1}
	This activity develops C1--C2 skills by identifying decision components and organizing consequences in payoff tables.
	
	\begin{enumerate}
		\item \textbf{Local Café Investment:} Identify 2 alternatives, 2 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Green Energy Production Mix:} Identify 3 alternatives, 3 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Global Shipping Network Design:} Identify 2 alternatives, 4 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Community Bakery Lunch Boxes:} Identify 2 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component: profit per item times number of items.
		\item \textbf{Rural Internet Subscriptions:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component: profit per item times number of items.
		\item \textbf{Coffee Export Routes:} Identify 2 alternatives, 2 states, and calculate payoff entries using a state-based variable component: profit per item times number of items.
		\item \textbf{Lake Ferry Tourism:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component: profit per item times number of items.
		\item \textbf{Cold-Storage Contracts:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component plus an alternative-based fixed component.
		\item \textbf{Solar Installation Bids:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component plus a state-based fixed component.
		\item \textbf{Air Cargo Charters:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component plus an alternative-based fixed component.
		\item \textbf{Seafood Export Licenses:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component plus a state-based fixed component.
	\end{enumerate}
