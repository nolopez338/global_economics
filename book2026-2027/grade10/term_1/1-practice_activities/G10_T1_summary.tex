	\section{G10\_T1\_L1\_C1-C2\_practice\_activity1}
	This activity develops C1--C2 skills by identifying decision components and organizing consequences in payoff tables.
	
	\begin{enumerate}
		\item \textbf{Local Café Investment:} Identify 2 alternatives, 2 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Green Energy Production Mix:} Identify 3 alternatives, 3 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Global Shipping Network Design:} Identify 2 alternatives, 4 states, and direct payoff entries for each alternative--state pair.
		\item \textbf{Community Bakery Lunch Boxes:} Identify 2 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component: profit per item times number of items.
		\item \textbf{Rural Internet Subscriptions:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component: profit per item times number of items.
		\item \textbf{Coffee Export Routes:} Identify 2 alternatives, 2 states, and calculate payoff entries using a state-based variable component: profit per item times number of items.
		\item \textbf{Lake Ferry Tourism:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component: profit per item times number of items.
		\item \textbf{Community Hall Emergency Rentals:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based fixed component only.
		\item \textbf{Mobile Clinic Service Contracts:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based fixed component only.
		\item \textbf{Harbor Inspection Agreements:} Identify 2 alternatives, 3 states, and calculate payoff entries using both a state-based fixed component and an alternative-based fixed component.
		\item \textbf{Cold-Storage Contracts:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component plus an alternative-based fixed component.
		\item \textbf{Solar Installation Bids:} Identify 3 alternatives, 2 states, and calculate payoff entries using an alternative-based variable component plus a state-based fixed component.
		\item \textbf{Air Cargo Charters:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component plus an alternative-based fixed component.
		\item \textbf{Seafood Export Licenses:} Identify 2 alternatives, 3 states, and calculate payoff entries using a state-based variable component plus a state-based fixed component.
	\end{enumerate}
	
	\section{G10\_T1\_L1\_C1-C2\_practice\_activity2}
	This activity extends C1--C2 skills by interpreting decision components and building payoff tables from revenue and cost information.

	\begin{enumerate}
		\item \textbf{Food Truck Menu Upgrade:} Identify 2 alternatives, 2 states, and calculate payoff entries as revenue minus cost by state.
		\item \textbf{School Fundraiser Ticket Plan:} Identify 2 alternatives, 3 states, and calculate payoff entries as revenue minus cost by attendance state.
		\item \textbf{City Bike-Share Pricing Strategy:} Identify 3 alternatives, 2 states, and calculate payoff entries as revenue minus cost by ridership state.
		\item \textbf{Subscription Bundle Choices:} Identify 3 alternatives, 2 states, and calculate payoff entries using subscription price times sign-ups minus state-based costs.
		\item \textbf{Regional Distribution Plan:} Identify 2 alternatives, 3 states, and calculate payoff entries using state-based revenue minus delivery-based variable costs.
		\item \textbf{Farmers Market Juice Stand:} Identify 2 alternatives, 2 states, and calculate payoff entries using state-based revenue minus production cost per bottle.
		\item \textbf{After-School Tutoring Sessions:} Identify 2 alternatives, 2 states, and calculate payoff entries using sessions delivered times net fee per session.
		\item \textbf{Art Fair Booth Options:} Identify 2 alternatives, 2 states, and calculate payoff entries using kits sold times price minus alternative-based fixed setup cost.
		\item \textbf{Emergency Supply Contract:} Identify 2 alternatives, 3 states, and calculate payoff entries using state-based retainer plus kit margin minus state-based compliance fee.
	\end{enumerate}
