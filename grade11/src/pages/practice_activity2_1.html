A regional logistics agency wants to estimate the mean time (in minutes) for cargo to clear port inspections.

Historical monitoring shows the population standard deviation is \(\sigma = 12\) minutes.

The agency wants a 95% confidence estimate with a maximum error of \(E = 3\) minutes.

Determine the minimum sample size needed.

↑

This solution establishes the sample size so the error will not exceed a specified amount by identifying \(E\), selecting \(z^*\), substituting into the formula, and rounding up to the minimum integer sample size.

Step 1: Identify the target error and confidence level. The maximum error is \(E = 3\) minutes and the confidence level is 95%, so \(z^* = 1.96\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{3}\right)^2 = \left(7.84\right)^2 = 61.47 \]

Step 3: Round up and verify Criterion 6. Because the sample size must ensure the error does not exceed \(E\), we round up: \(n = 62\). With \(n=62\), the margin of error is at most 3 minutes, satisfying Criterion 6.

A development agency wants to estimate the mean monthly food spending for households in a region.

A pilot survey suggests the population standard deviation is about \(\sigma \approx 45\) currency units.

The agency wants a 90% confidence estimate with a maximum error of \(E = 8\) currency units.

Determine the minimum sample size needed.

↑

This solution establishes the sample size so the error will not exceed a specified amount by stating \(E\), selecting the correct \(z^*\), substituting into the formula, and rounding up to the minimum safe integer.

Step 1: Identify the target error and confidence level. The maximum error is \(E = 8\) and the confidence level is 90%, so \(z^* = 1.645\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.645 \cdot 45}{8}\right)^2 = \left(9.253\right)^2 \approx 85.62 \]

Step 3: Round up and verify Criterion 6. We must round up to keep the error within \(E\), so \(n = 86\). This minimum sample size guarantees the margin of error is no more than 8 currency units.

A trade corridor study tracks the mean daily transit time (in minutes) for cross-border trucks.

A pilot sample suggests the population standard deviation is about \(\sigma \approx 25\) minutes.

The planners want a 95% confidence estimate with a maximum error of \(E = 4\) minutes.

Determine the minimum sample size needed.

↑

This solution establishes the sample size so the error will not exceed a specified amount by stating \(E\), identifying the confidence level and \(z\), and rounding up to the minimum whole number.

Step 1: Identify the target error and confidence level. The maximum error is \(E = 4\) minutes and the confidence level is 95%, so \(z^* = 1.96\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 25}{4}\right)^2 = \left(12.25\right)^2 = 150.06 \]

Step 3: Round up and verify Criterion 6. We round up to keep the margin of error within \(E\), so \(n = 151\). A sample of 151 trucks ensures the error is no more than 4 minutes.

An inflation monitoring team wants to estimate the mean price of a staple food basket (in dollars) across markets.

They know the population standard deviation is \(\sigma = 0.60\) dollars.

The team wants a 90% confidence estimate with a maximum error of \(E = 0.15\) dollars.

Determine the minimum sample size needed.

↑

This solution establishes the sample size so the error will not exceed a specified amount by specifying \(E\), using the correct \(z^*\) value, calculating \(n\), and rounding up.

Step 1: Identify the target error and confidence level. The maximum error is \(E = 0.15\) dollars and the confidence level is 90%, so \(z^* = 1.645\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.645 \cdot 0.60}{0.15}\right)^2 = \left(6.58\right)^2 = 43.30 \]

Step 3: Round up and verify Criterion 6. We round up to ensure the error stays within \(E\), so \(n = 44\). Sampling 44 markets keeps the margin of error at or below 0.15 dollars.
