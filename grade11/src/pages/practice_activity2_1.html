A regional logistics agency wants to estimate the mean time (in minutes) for cargo to clear port inspections. Historical monitoring shows the population standard deviation is \(\sigma = 12\) minutes. The agency wants a 95% confidence estimate with a maximum error of \(E = 3\) minutes. Determine the minimum sample size needed.

↑

This solution establishes the sample size so the error will not exceed a specified amount by identifying \(E\), selecting \(z^*\), substituting into the formula, and rounding up to the minimum integer sample size.

Step 1: Identify the target error and confidence level. The maximum error is \(E = 3\) minutes and the confidence level is 95%, so \(z^* = 1.96\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{3}\right)^2 = \left(7.84\right)^2 = 61.47 \]

Step 3: Round up and verify Criterion 6. Because the sample size must ensure the error does not exceed \(E\), we round up: \(n = 62\). With \(n=62\), the margin of error is at most 3 minutes, satisfying Criterion 6.

A development agency wants to estimate the mean monthly food spending for households in a region. A pilot survey suggests the population standard deviation is about \(\sigma \approx 45\) currency units. The agency wants a 90% confidence estimate with a target interval width of \(W = 16\) currency units. The width is defined as \(W = \text{Upper} - \text{Lower} = 2E\). Determine the minimum sample size needed.

↑

This solution establishes the sample size so the interval width will not exceed a specified amount by converting \(W\) to \(E\), selecting the correct \(z^*\), substituting into the formula, and rounding up to the minimum safe integer.

Step 1: Identify the target width and confidence level. The target width is \(W = 16\), so \(E = W/2 = 8\). The confidence level is 90%, so \(z^* = 1.645\).

Step 2: Apply the sample size formula.

\[ n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.645 \cdot 45}{8}\right)^2 = \left(9.253\right)^2 \approx 85.62 \]

Step 3: Round up and verify Criterion 6. We must round up to keep the width within \(W\), so \(n = 86\). This minimum sample size guarantees the interval width is no more than 16 currency units.

A shipping cooperative wants to estimate the mean fuel cost per container on a trade route. Company records indicate the population standard deviation is \(\sigma = 18\) dollars. The cooperative wants the maximum margin of error to be no more than \(E = 5\) dollars. Determine the minimum sample size needed for 90%, 95%, and 98% confidence.

↑

This solution calculates the minimum sample size needed to keep the margin of error at or below \(E = 5\) dollars for three confidence levels.

Step 1: Identify the confidence levels and \(z^*\) values. Use \(z^* = 1.645\) for 90%, \(z^* = 1.96\) for 95%, and \(z^* = 2.326\) for 98% confidence.

Step 2: Apply the sample size formula for each confidence level.

\[ \text{90\%: } n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.645 \cdot 18}{5}\right)^2 = \left(5.922\right)^2 \approx 35.07 \]

\[ \text{95\%: } n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 18}{5}\right)^2 = \left(7.056\right)^2 \approx 49.79 \]

\[ \text{98\%: } n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{2.326 \cdot 18}{5}\right)^2 = \left(8.374\right)^2 \approx 70.12 \]

Step 3: Round up to ensure the error does not exceed 5 dollars. The required sample sizes are \(n = 36\) for 90%, \(n = 50\) for 95%, and \(n = 71\) for 98% confidence.

↑

This subsection highlights Criterion 6 by linking each confidence level to the minimum sample size that keeps the error at or below \(E = 5\) dollars.

Step 1: State the rounded minimum sample sizes. After rounding up, the cooperative needs \(n = 36\) for 90% confidence, \(n = 50\) for 95% confidence, and \(n = 71\) for 98% confidence.

Step 2: Conclude with the planning decision. Selecting the sample size tied to the desired confidence level guarantees the margin of error will not exceed 5 dollars.

A trade corridor study tracks the mean daily transit time (in minutes) for cross-border trucks. The population standard deviation is known to be \(\sigma = 12\) minutes. The planners will use a 95% confidence level. Determine the minimum sample size needed for each target margin of error: \(E = 3\) minutes, \(E = 4\) minutes, and \(E = 5\) minutes.

↑

This solution determines the minimum sample size for a fixed 95% confidence level with three different target margins of error.

Step 1: Identify the confidence level and \(z^*\). For 95% confidence, \(z^* = 1.96\).

Step 2: Apply the sample size formula for each margin of error.

\[ \text{For } E = 3: \quad n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{3}\right)^2 = \left(7.84\right)^2 = 61.47 \]

\[ \text{For } E = 4: \quad n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{4}\right)^2 = \left(5.88\right)^2 = 34.57 \]

\[ \text{For } E = 5: \quad n = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{5}\right)^2 = \left(4.704\right)^2 = 22.13 \]

Step 3: Round up to ensure each error target is met. The minimum sample sizes are \(n = 62\) for \(E = 3\), \(n = 35\) for \(E = 4\), and \(n = 23\) for \(E = 5\).

↑

This subsection connects the planning decision to Criterion 6 by emphasizing how each target error determines the minimum sample size.

Step 1: Match each margin of error to its rounded sample size. At 95% confidence, use \(n = 62\) for \(E = 3\) minutes, \(n = 35\) for \(E = 4\) minutes, and \(n = 23\) for \(E = 5\) minutes.

Step 2: Conclude the sampling plan. Choosing the sample size tied to the desired error ensures the margin of error will not exceed the stated target at 95% confidence.

An inflation monitoring team wants to estimate the mean price of a staple food basket (in dollars) across markets. They know the population standard deviation is \(\sigma = 0.60\) dollars and the current sample mean is \(\bar{x} = 2.85\) dollars. The team has collected \(n_{\text{current}} = 36\) market prices and wants a 90% confidence interval. (a) Construct the confidence interval for the mean price. (b) How many additional observations are required to reduce the margin of error to \(E = 0.12\) dollars?

↑

This solution calculates the confidence interval for the mean with known variance using the current sample information.

Step 1: Compute the standard error.

\[ SE = \frac{\sigma}{\sqrt{n_{\text{current}}}} = \frac{0.60}{\sqrt{36}} = 0.10 \]

Step 2: Construct the confidence interval. For 90% confidence, \(z^* = 1.645\).

\[ \bar{x} \pm z^* SE = 2.85 \pm 1.645(0.10) = 2.85 \pm 0.16 = (2.69,\ 3.01) \]

Conclusion. The 90% confidence interval is \((2.69,\ 3.01)\) dollars.

↑

This subsection establishes the required sample size so the margin of error does not exceed the target amount.

Step 1: Use the target error and confidence level. The target margin of error is \(E = 0.12\) dollars and for 90% confidence, \(z^* = 1.645\).

Step 2: Compute the required sample size.

\[ n_{\text{required}} = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.645 \cdot 0.60}{0.12}\right)^2 = \left(8.225\right)^2 \approx 67.65 \]

Step 3: Determine additional observations. Round up to \(n_{\text{required}} = 68\). The additional observations needed are \(68 - 36 = 32\).

Conclusion. The team needs 32 more observations so the error will not exceed 0.12 dollars.

A distribution center tracks the mean time (in minutes) required to handle incoming shipments.

The population standard deviation is known to be \(\sigma = 12\) minutes.

The current sample mean is \(\bar{x} = 52\) minutes with \(n_{\text{current}} = 40\) observations and a 95% confidence level.

(a) Construct the confidence interval for the mean handling time.

(b) How many additional observations are needed to reach a target interval width of \(W = 6\) minutes? (Recall \(W = \text{Upper} - \text{Lower} = 2E\).)

↑

This solution calculates a confidence interval with known variance using the current sample information.

Step 1: Compute the standard error.

\[ SE = \frac{\sigma}{\sqrt{n_{\text{current}}}} = \frac{12}{\sqrt{40}} \approx 1.90 \]

Step 2: Construct the confidence interval. For 95% confidence, \(z^* = 1.96\).

\[ \bar{x} \pm z^* SE = 52 \pm 1.96(1.90) = 52 \pm 3.72 = (48.28,\ 55.72) \]

Conclusion. The 95% confidence interval is \((48.28,\ 55.72)\) minutes.

↑

This subsection establishes the required sample size so the interval width stays within the target amount.

Step 1: Convert width to margin of error. The target width is \(W = 6\), so \(E = W/2 = 3\).

Step 2: Compute the required sample size. For 95% confidence, \(z^* = 1.96\).

\[ n_{\text{required}} = \left(\frac{z^*\,\sigma}{E}\right)^2 = \left(\frac{1.96 \cdot 12}{3}\right)^2 = \left(7.84\right)^2 \approx 61.47 \]

Step 3: Determine additional observations. Round up to \(n_{\text{required}} = 62\). The additional observations needed are \(62 - 40 = 22\).

Conclusion. The distribution center needs 22 more observations so the error will not exceed 3 minutes (width 6 minutes).

A microcredit program tracks the mean processing fee (in dollars) charged per loan.

The population standard deviation is known to be \(\sigma = 16\) dollars.

The current sample mean is \(\bar{x} = 88\) dollars with \(n_{\text{current}} = 64\) observations at 90% confidence.

(a) Construct the confidence interval for the mean fee.

(b) How many observations are required so that the new margin of error is the current margin of error minus 1.0 dollar? (Use \(E_{\text{new}} = E_{\text{current}} - 1.0\).)

↑

This solution computes the confidence interval and then checks whether additional observations are needed for a larger margin of error.

Step 1: Compute the standard error.

\[ SE = \frac{\sigma}{\sqrt{n_{\text{current}}}} = \frac{16}{\sqrt{64}} = 2 \]

Step 2: Construct the confidence interval. For 90% confidence, \(z^* = 1.645\).

\[ \bar{x} \pm z^* SE = 88 \pm 1.645(2) = 88 \pm 3.29 = (84.71,\ 91.29) \]

Step 3: Compute the required sample size for the smaller margin of error. The current margin of error is \(E_{\text{current}} = 3.29\), so the new target is \(E_{\text{new}} = 3.29 - 1.0 = 2.29\).

\[ n_{\text{required}} = \left(\frac{z^*\,\sigma}{E_{\text{new}}}\right)^2 = \left(\frac{1.645 \cdot 16}{2.29}\right)^2 = \left(11.50\right)^2 \approx 132.25 \]

Step 4: Determine additional observations. Round up to \(n_{\text{required}} = 133\). The additional observations needed are \(133 - 64 = 69\).

Conclusion. The 90% confidence interval is \((84.71,\ 91.29)\), and a total of 133 observations are required to reduce the margin of error by 1.0 dollar.

A rural water agency measures the mean daily delivery volume (in liters) for tanker trucks.

The population standard deviation is \(\sigma = 18\) liters.

The current sample mean is \(\bar{x} = 205\) liters with \(n_{\text{current}} = 81\) observations at 95% confidence.

(a) Construct the confidence interval for the mean delivery volume.

(b) How many observations are required so that the new margin of error is the current margin of error minus 1.0 liter? (Use \(E_{\text{new}} = E_{\text{current}} - 1.0\).)

↑

This solution computes the confidence interval and then checks whether additional observations are needed for a larger interval width.

Step 1: Compute the standard error.

\[ SE = \frac{\sigma}{\sqrt{n_{\text{current}}}} = \frac{18}{\sqrt{81}} = 2 \]

Step 2: Construct the confidence interval. For 95% confidence, \(z^* = 1.96\).

\[ \bar{x} \pm z^* SE = 205 \pm 1.96(2) = 205 \pm 3.92 = (201.08,\ 208.92) \]

Step 3: Compute the required sample size for the smaller margin of error. The current margin of error is \(E_{\text{current}} = 3.92\), so the new target is \(E_{\text{new}} = 3.92 - 1.0 = 2.92\).

\[ n_{\text{required}} = \left(\frac{z^*\,\sigma}{E_{\text{new}}}\right)^2 = \left(\frac{1.96 \cdot 18}{2.92}\right)^2 = \left(12.08\right)^2 \approx 145.93 \]

Step 4: Determine additional observations. Round up to \(n_{\text{required}} = 146\). The additional observations needed are \(146 - 81 = 65\).

Conclusion. The 95% confidence interval is \((201.08,\ 208.92)\), and a total of 146 observations are required to reduce the margin of error by 1.0 liter.

An export office tracks the mean documentation cost (in dollars) per shipment.

The population standard deviation is \(\sigma = 21\) dollars.

The current sample mean is \(\bar{x} = 310\) dollars with \(n_{\text{current}} = 49\) observations at 98% confidence.

(a) Construct the confidence interval for the mean documentation cost.

(b) How many observations are required so that the new margin of error is the current margin of error minus 1.5 dollars? (Use \(E_{\text{new}} = E_{\text{current}} - 1.5\).)

↑

This solution computes the confidence interval and then checks whether additional observations are needed for a larger margin of error.

Step 1: Compute the standard error.

\[ SE = \frac{\sigma}{\sqrt{n_{\text{current}}}} = \frac{21}{\sqrt{49}} = 3 \]

Step 2: Construct the confidence interval. For 98% confidence, \(z^* = 2.326\).

\[ \bar{x} \pm z^* SE = 310 \pm 2.326(3) = 310 \pm 6.98 = (303.02,\ 316.98) \]

Step 3: Compute the required sample size for the smaller margin of error. The current margin of error is \(E_{\text{current}} = 6.98\), so the new target is \(E_{\text{new}} = 6.98 - 1.5 = 5.48\).

\[ n_{\text{required}} = \left(\frac{z^*\,\sigma}{E_{\text{new}}}\right)^2 = \left(\frac{2.326 \cdot 21}{5.48}\right)^2 = \left(8.91\right)^2 \approx 79.40 \]

Step 4: Determine additional observations. Round up to \(n_{\text{required}} = 80\). The additional observations needed are \(80 - 49 = 31\).

Conclusion. The 98% confidence interval is \((303.02,\ 316.98)\), and a total of 80 observations are required to reduce the margin of error by 1.5 dollars.
