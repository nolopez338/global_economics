A benchmark claim concerns a population proportion. A random sample of size \(n\) has \(x\) successes, so \(\hat p=x/n\). At significance level \(\alpha\), test whether the true proportion is lower than, higher than, or different from the benchmark \(p_0\).

Questions

  1. Identify the population parameter and write hypotheses for lower-tail, higher-tail, and two-tailed alternatives.
  2. State the condition checks for a one-sample proportion \(z\)-test.
  3. Write the one-sample proportion test statistic.
  4. State how the p-value is computed for lower-tail, higher-tail, and two-tailed tests.
  5. State the decision rule and write a contextual conclusion template.
  • Parameter: \(p\) = true population proportion for the context.
  • Hypotheses: \(H_0\colon p=p_0,\quad H_a\colon p<p_0\ \text{or}\ p>p_0\ \text{or}\ p\ne p_0.\)
  • Conditions: Check \(np_0\ge 10\) and \(n(1-p_0)\ge 10\).
  • Test statistic: \(z=\dfrac{\hat p-p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}.\)
  • p-value by tail type:
    • Lower-tail: \(P(Z\le z_{obs})\)
    • Higher-tail: \(P(Z\ge z_{obs})\)
    • Two-tailed: \(2P(Z\ge |z_{obs}|)\)
  • Decision rule: Reject \(H_0\) if p-value \(\le \alpha\); otherwise fail to reject \(H_0\).
  • Conclusion template: “There is sufficient evidence” (reject \(H_0\)) or “There is not enough evidence” (fail to reject \(H_0\)) for the contextual claim in \(H_a\).

A school reports that 72% of assignments are submitted on time. In a random sample of 250 assignments, 170 are submitted on time. At \(\alpha=5\%\), test whether the true on-time proportion is lower than 72%.

  • Parameter: \(p\) = true on-time submission proportion.
  • Hypotheses: \(H_0\colon p=0.72\), \(H_a\colon p<0.72\).
  • Conditions: \(np_0=250(0.72)=180\ge10\), \(n(1-p_0)=250(0.28)=70\ge10\).
  • Sample proportion: \(\hat p=170/250=0.68\).
  • Test statistic: \(z=\dfrac{0.68-0.72}{\sqrt{\frac{0.72(0.28)}{250}}}\approx -1.41\).
  • p-value: \(P(Z\le -1.41)\approx 0.0793\).
  • Decision: Fail to reject \(H_0\) since \(0.0793>0.05\).
  • Conclusion: There is not enough evidence that the true on-time submission rate is below 72%.

A district dashboard states that 40% of teachers regularly use a new digital platform. After training, a random sample of 300 teachers shows 138 regular users. At \(\alpha=5\%\), test whether the true adoption proportion is higher than 40%.

  • Parameter: \(p\) = true regular-use proportion.
  • Hypotheses: \(H_0\colon p=0.40\), \(H_a\colon p>0.40\).
  • Conditions: \(np_0=120\ge10\), \(n(1-p_0)=180\ge10\).
  • Sample proportion: \(\hat p=138/300=0.46\).
  • Test statistic: \(z=\dfrac{0.46-0.40}{\sqrt{\frac{0.40(0.60)}{300}}}\approx 2.12\).
  • p-value: \(P(Z\ge 2.12)\approx 0.0170\).
  • Decision: Reject \(H_0\) since \(0.0170<0.05\).
  • Conclusion: There is sufficient evidence that the true platform-adoption rate is above 40%.

A quality team states that 85% of shifts complete a safety checklist. In a random sample of 320 shifts, 256 completed the checklist. At \(\alpha=5\%\), test whether the true completion proportion is different from 85%.

  • Parameter: \(p\) = true checklist-completion proportion.
  • Hypotheses: \(H_0\colon p=0.85\), \(H_a\colon p\ne0.85\).
  • Conditions: \(np_0=272\ge10\), \(n(1-p_0)=48\ge10\).
  • Sample proportion: \(\hat p=256/320=0.80\).
  • Test statistic: \(z=\dfrac{0.80-0.85}{\sqrt{\frac{0.85(0.15)}{320}}}\approx -2.50\).
  • p-value: \(2P(Z\le -2.50)\approx 0.0124\).
  • Decision: Reject \(H_0\) since \(0.0124<0.05\).
  • Conclusion: There is sufficient evidence that the true checklist-compliance rate differs from 85%.

A training unit expects 60% completion of optional modules. In a random sample of 500 workers, 285 complete the module. Test whether the true completion rate is lower than 60% at \(\alpha=5\%\) and \(\alpha=1\%\).

  • Hypotheses: \(H_0\colon p=0.60\), \(H_a\colon p<0.60\).
  • Conditions: \(np_0=300\ge10\), \(n(1-p_0)=200\ge10\).
  • Sample proportion: \(\hat p=285/500=0.57\).
  • Test statistic: \(z=\dfrac{0.57-0.60}{\sqrt{\frac{0.60(0.40)}{500}}}\approx -1.37\).
  • p-value: \(P(Z\le -1.37)\approx 0.0853\).
  • Decision at 5%: Fail to reject \(H_0\) since \(0.0853>0.05\).
  • Decision at 1%: Fail to reject \(H_0\) since \(0.0853>0.01\).
  • Sensitivity note: The sample evidence is fixed, and both thresholds lead to the same non-rejection outcome.

A program benchmark is 70% attendance. In a random sample of 200 participants, 128 attend. Use \(\alpha=5\%\).

Questions

  1. Test \(H_a\colon p<0.70\).
  2. Test \(H_a\colon p>0.70\).

Common setup

  • \(\hat p=128/200=0.64\), \(p_0=0.70\).
  • Conditions: \(np_0=140\ge10\), \(n(1-p_0)=60\ge10\).
  • \(z=\dfrac{0.64-0.70}{\sqrt{\frac{0.70(0.30)}{200}}}\approx -1.85\).

Part (a): Lower-tail

  • p-value: \(P(Z\le -1.85)\approx 0.0322\).
  • Decision: Reject \(H_0\) since \(0.0322<0.05\).

Part (b): Higher-tail

  • p-value: \(P(Z\ge -1.85)\approx 0.9678\).
  • Decision: Fail to reject \(H_0\) since \(0.9678>0.05\).

Comparison note: Changing only the alternative changes tail type, p-value, and decision.

A call center benchmark is 70% first-call resolution. In a random sample of 400 calls, 308 are resolved on first call. Use \(\alpha=5\%\), and also re-check the two-tailed result at \(\alpha=1\%\).

Questions

  1. Test \(H_a\colon p>0.70\).
  2. Test \(H_a\colon p<0.70\).
  3. Test \(H_a\colon p\ne0.70\) at 5%.
  4. Re-evaluate part (c) at 1%.
  • \(\hat p=308/400=0.77\), \(p_0=0.70\).
  • Conditions: \(np_0=280\ge10\), \(n(1-p_0)=120\ge10\).
  • \(z=\dfrac{0.77-0.70}{\sqrt{\frac{0.70(0.30)}{400}}}\approx 2.89\).
  • Higher-tail p-value \(\approx 0.0019\), reject \(H_0\).
  • Lower-tail p-value \(\approx 0.9981\), fail to reject \(H_0\).
  • Two-tailed p-value \(\approx 0.0038\).
  • Two-tailed decision at 5%: reject \(H_0\).
  • Two-tailed decision at 1%: reject \(H_0\).
  • Conclusion: Strong evidence in the higher direction; both higher-tail and two-tailed tests reject, while the opposite direction does not.
