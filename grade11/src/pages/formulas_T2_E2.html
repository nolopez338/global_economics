\[ \hat{p} = \frac{x}{n} \]

Where:

  • \(\hat{p}\): sample proportion (point estimate).
  • \(x\): number of successes in the sample.
  • \(n\): sample size.
↑

\[ SE(\hat{p}) = \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

Where:

  • \(SE(\hat{p})\): standard error of the sample proportion.
  • \(\hat{p}\): sample proportion.
  • \(n\): sample size.
↑

\[ E = z^* \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

Where:

  • \(E\): margin of error.
  • \(z^*\): critical value for the chosen confidence level.
  • \(\hat{p}\): sample proportion.
  • \(n\): sample size.
↑

\[ \hat{p} \pm z^* \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

Where:

  • \(\hat{p}\): sample proportion.
  • \(z^*\): critical value for the chosen confidence level.
  • \(n\): sample size.
↑

\[ n \ge \frac{(z^*)^2 p^*(1-p^*)}{E^2} \]

Where:

  • \(n\): minimum required sample size.
  • \(z^*\): critical value for the chosen confidence level.
  • \(p^*\): planning value for the population proportion.
  • \(E\): desired margin of error.

Note: Round up \(n\) to the next whole number.

↑

\[ n = \left(\frac{z^* \sigma}{E}\right)^2 \]

Where:

  • \(n\): required sample size.
  • \(z^*\): critical value for the chosen confidence level.
  • \(\sigma\): population standard deviation (known).
  • \(E\): desired margin of error.

Note: Round up \(n\) to the next whole number.
