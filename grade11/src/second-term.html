Confidence intervals estimate a population value (like a mean or proportion) using a range of plausible values based on a sample.

  • Margin of error: The distance from the sample estimate to each end of the interval. It tells us how much “wiggle room” we allow for sampling uncertainty.
  • Confidence level: The long-run success rate of the method. For example, a 95% confidence interval means that if we repeated the sampling process many times, about 95% of the intervals would contain the true population value.
  • Interpreting the interval: We do not say there is a 95% chance the parameter is in this specific interval. Instead, we say we are using a method that captures the true value 95% of the time.

What makes an interval wider or narrower?

  • Larger samples (\(n\)): Decrease the standard error, so intervals become narrower and more precise.
  • Higher confidence levels: Require larger critical values (like \(z_{\alpha/2}\) or \(t_{\alpha/2}\)), which makes intervals wider.
  • More variability: Larger standard deviation increases the margin of error, so intervals widen.

In short: bigger samples shrink intervals, while higher confidence levels widen them. Good statistical decisions balance precision with reliability.
